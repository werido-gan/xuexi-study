### 26.2 文本预处理

文本预处理是 NLP 任务的重要步骤，它可以提高模型的性能。

#### 26.2.1 文本清洗

```python
import re
import string

def clean_text(text):
    # 转换为小写
    text = text.lower()
    
    # 移除标点符号
    text = text.translate(str.maketrans('', '', string.punctuation))
    
    # 移除数字
    text = re.sub(r'\d+', '', text)
    
    # 移除多余的空格
    text = re.sub(r'\s+', ' ', text).strip()
    
    return text

# 测试
text = "Hello, World! 123 This is a test."
print(f"Original: {text}")
print(f"Cleaned: {clean_text(text)}")
```

#### 26.2.2 分词

分词是将文本分割成词语或标记的过程。

```python
import nltk
from nltk.tokenize import word_tokenize, sent_tokenize

# 下载必要的资源
nltk.download('punkt')

# 句子分词
sentences = sent_tokenize("Hello world. This is a test.")
print(f"Sentences: {sentences}")

# 词语分词
words = word_tokenize("Hello world. This is a test.")
print(f"Words: {words}")

# 使用 spaCy 进行分词
import spacy

nlp = spacy.load('en_core_web_sm')
doc = nlp("Hello world. This is a test.")
tokens = [token.text for token in doc]
print(f"SpaCy tokens: {tokens}")
```

#### 26.2.3 停用词移除

停用词是指在文本中频繁出现但对语义贡献不大的词语，如 "the"、"is"、"and" 等。

```python
import nltk
from nltk.corpus import stopwords

# 下载停用词
nltk.download('stopwords')

# 获取英文停用词
stop_words = set(stopwords.words('english'))

# 移除停用词
def remove_stopwords(text):
    words = word_tokenize(text)
    filtered_words = [word for word in words if word.lower() not in stop_words]
    return ' '.join(filtered_words)

# 测试
text = "This is a test sentence with some stop words."
print(f"Original: {text}")
print(f"Without stopwords: {remove_stopwords(text)}")

# 使用 spaCy 移除停用词
import spacy

nlp = spacy.load('en_core_web_sm')
def remove_stopwords_spacy(text):
    doc = nlp(text)
    filtered_words = [token.text for token in doc if not token.is_stop]
    return ' '.join(filtered_words)

print(f"Without stopwords (spaCy): {remove_stopwords_spacy(text)}")
```

#### 26.2.4 词干提取和词形还原

- **词干提取**：将词语还原为词干形式（如 "running" → "run"）
- **词形还原**：将词语还原为基本形式（如 "better" → "good"）

```python
import nltk
from nltk.stem import PorterStemmer, WordNetLemmatizer

# 下载必要的资源
nltk.download('wordnet')

# 词干提取
stemmer = PorterStemmer()
words = ["running", "ran", "runs", "better", "best"]
stemmed_words = [stemmer.stem(word) for word in words]
print(f"Stemmed words: {stemmed_words}")

# 词形还原
lemmatizer = WordNetLemmatizer()
lemmatized_words = [lemmatizer.lemmatize(word) for word in words]
print(f"Lemmatized words: {lemmatized_words}")

# 使用 spaCy 进行词形还原
import spacy

nlp = spacy.load('en_core_web_sm')
doc = nlp("running ran runs better best")
lemmatized_spacy = [token.lemma_ for token in doc]
print(f"Lemmatized with spaCy: {lemmatized_spacy}")
```

### 26.3 文本表示

文本表示是将文本转换为数字向量的过程，以便计算机能够处理。

#### 26.3.1 词袋模型

词袋模型是一种简单的文本表示方法，它将文本表示为词语的频率向量。

```python
from sklearn.feature_extraction.text import CountVectorizer

# 语料库
corpus = [
    "This is a test sentence.",
    "This is another test sentence.",
    "I am testing the CountVectorizer."
]

# 创建词袋模型
vectorizer = CountVectorizer()
X = vectorizer.fit_transform(corpus)

# 查看词汇表
print(f"Vocabulary: {vectorizer.vocabulary_}")

# 查看向量表示
print(f"Vector representation:\n{X.toarray()}")

# 新文本的向量表示
new_text = ["This is a new test sentence."]
X_new = vectorizer.transform(new_text)
print(f"New text vector:\n{X_new.toarray()}")
```

#### 26.3.2 TF-IDF

TF-IDF（Term Frequency-Inverse Document Frequency）是一种更复杂的文本表示方法，它考虑了词语在文档中的频率和在整个语料库中的稀有程度。

```python
from sklearn.feature_extraction.text import TfidfVectorizer

# 语料库
corpus = [
    "This is a test sentence.",
    "This is another test sentence.",
    "I am testing the TfidfVectorizer."
]

# 创建 TF-IDF 模型
vectorizer = TfidfVectorizer()
X = vectorizer.fit_transform(corpus)

# 查看词汇表
print(f"Vocabulary: {vectorizer.vocabulary_}")

# 查看向量表示
print(f"TF-IDF vector representation:\n{X.toarray()}")

# 新文本的向量表示
new_text = ["This is a new test sentence."]
X_new = vectorizer.transform(new_text)
print(f"New text TF-IDF vector:\n{X_new.toarray()}")
```

#### 26.3.3 Word2Vec

Word2Vec 是一种将词语表示为低维向量的方法，它能够捕捉词语之间的语义关系。

```python
from gensim.models import Word2Vec
import nltk
from nltk.tokenize import word_tokenize

# 语料库
corpus = [
    "This is a test sentence.",
    "This is another test sentence.",
    "I am testing Word2Vec.",
    "Word2Vec is a word embedding technique."
]

# 分词
tokenized_corpus = [word_tokenize(sentence.lower()) for sentence in corpus]

# 训练 Word2Vec 模型
model = Word2Vec(tokenized_corpus, vector_size=100, window=5, min_count=1, workers=4)

# 获取词语向量
word_vector = model.wv['test']
print(f"Word vector for 'test': {word_vector}")

# 寻找相似的词语
similar_words = model.wv.most_similar('test')
print(f"Words similar to 'test': {similar_words}")

# 计算词语之间的相似度
similarity = model.wv.similarity('test', 'testing')
print(f"Similarity between 'test' and 'testing': {similarity}")
```

#### 26.3.4 GloVe

GloVe（Global Vectors for Word Representation）是另一种词嵌入方法，它结合了词袋模型和 Word2Vec 的优点。

```python
from gensim.models import KeyedVectors

# 加载预训练的 GloVe 模型
# 下载地址：https://nlp.stanford.edu/projects/glove/
model = KeyedVectors.load_word2vec_format('glove.6B.100d.txt', binary=False, no_header=True)

# 获取词语向量
word_vector = model['test']
print(f"Word vector for 'test': {word_vector}")

# 寻找相似的词语
similar_words = model.most_similar('test')
print(f"Words similar to 'test': {similar_words}")

# 计算词语之间的相似度
similarity = model.similarity('test', 'testing')
print(f"Similarity between 'test' and 'testing': {similarity}")
```

#### 26.3.5 BERT 嵌入

BERT（Bidirectional Encoder Representations from Transformers）是一种基于 Transformer 的预训练语言模型，它能够捕捉双向上下文信息。

```python
from transformers import BertTokenizer, BertModel
import torch

# 加载预训练的 BERT 模型和分词器
tokenizer = BertTokenizer.from_pretrained('bert-base-uncased')
model = BertModel.from_pretrained('bert-base-uncased')

# 文本
text = "This is a test sentence."

# 分词
inputs = tokenizer(text, return_tensors='pt')

# 获取嵌入
with torch.no_grad():
    outputs = model(**inputs)

# 最后一层的隐藏状态
last_hidden_state = outputs.last_hidden_state
print(f"Last hidden state shape: {last_hidden_state.shape}")

# [CLS] 标记的嵌入（用于句子级任务）
cls_embedding = last_hidden_state[:, 0, :]
print(f"CLS embedding shape: {cls_embedding.shape}")
```

### 26.4 文本分类

文本分类是将文本分类到预定义的类别中的任务。

#### 26.4.1 使用传统机器学习模型

```python
from sklearn.datasets import fetch_20newsgroups
from sklearn.feature_extraction.text import TfidfVectorizer
from sklearn.model_selection import train_test_split
from sklearn.naive_bayes import MultinomialNB
from sklearn.metrics import accuracy_score, classification_report

# 加载数据
data = fetch_20newsgroups(subset='all', categories=['alt.atheism', 'comp.graphics', 'sci.med', 'soc.religion.christian'])

# 分割数据
X_train, X_test, y_train, y_test = train_test_split(data.data, data.target, test_size=0.2, random_state=42)

# 特征提取
vectorizer = TfidfVectorizer(stop_words='english')
X_train_vec = vectorizer.fit_transform(X_train)
X_test_vec = vectorizer.transform(X_test)

# 训练模型
model = MultinomialNB()
model.fit(X_train_vec, y_train)

# 预测
y_pred = model.predict(X_test_vec)

# 评估
accuracy = accuracy_score(y_test, y_pred)
print(f"Accuracy: {accuracy}")
print("\nClassification Report:")
print(classification_report(y_test, y_pred, target_names=data.target_names))
```

#### 26.4.2 使用深度学习模型

```python
import torch
import torch.nn as nn
import torch.optim as optim
from sklearn.datasets import fetch_20newsgroups
from sklearn.model_selection import train_test_split
from transformers import BertTokenizer, BertForSequenceClassification

# 加载数据
data = fetch_20newsgroups(subset='all', categories=['alt.atheism', 'comp.graphics', 'sci.med', 'soc.religion.christian'])

# 分割数据
X_train, X_test, y_train, y_test = train_test_split(data.data, data.target, test_size=0.2, random_state=42)

# 加载预训练的 BERT 模型和分词器
tokenizer = BertTokenizer.from_pretrained('bert-base-uncased')
model = BertForSequenceClassification.from_pretrained('bert-base-uncased', num_labels=len(data.target_names))

# 准备数据
train_encodings = tokenizer(X_train, truncation=True, padding=True, max_length=128)
test_encodings = tokenizer(X_test, truncation=True, padding=True, max_length=128)

# 创建数据集
class NewsDataset(torch.utils.data.Dataset):
    def __init__(self, encodings, labels):
        self.encodings = encodings
        self.labels = labels
    
    def __getitem__(self, idx):
        item = {key: torch.tensor(val[idx]) for key, val in self.encodings.items()}
        item['labels'] = torch.tensor(self.labels[idx])
        return item
    
    def __len__(self):
        return len(self.labels)

train_dataset = NewsDataset(train_encodings, y_train)
test_dataset = NewsDataset(test_encodings, y_test)

# 创建数据加载器
train_loader = torch.utils.data.DataLoader(train_dataset, batch_size=16, shuffle=True)
test_loader = torch.utils.data.DataLoader(test_dataset, batch_size=16, shuffle=False)

# 定义优化器
optimizer = optim.AdamW(model.parameters(), lr=5e-5)

# 训练模型
device = torch.device('cuda' if torch.cuda.is_available() else 'cpu')
model.to(device)

for epoch in range(3):
    model.train()
    for batch in train_loader:
        batch = {k: v.to(device) for k, v in batch.items()}
        outputs = model(**batch)
        loss = outputs.loss
        loss.backward()
        optimizer.step()
        optimizer.zero_grad()
    
    print(f"Epoch {epoch+1} completed")

# 评估模型
model.eval()
y_pred = []
y_true = []

with torch.no_grad():
    for batch in test_loader:
        batch = {k: v.to(device) for k, v in batch.items()}
        outputs = model(**batch)
        logits = outputs.logits
        predictions = torch.argmax(logits, dim=-1)
        y_pred.extend(predictions.cpu().tolist())
        y_true.extend(batch['labels'].cpu().tolist())

# 计算准确率
accuracy = accuracy_score(y_true, y_pred)
print(f"Accuracy: {accuracy}")
print("\nClassification Report:")
print(classification_report(y_true, y_pred, target_names=data.target_names))
```

### 26.5 情感分析

情感分析是分析文本情感倾向（如积极、消极、中性）的任务。

```python
from transformers import pipeline

# 加载情感分析管道
sentiment_analysis = pipeline('sentiment-analysis')

# 测试文本
texts = [
    "I love this movie! It's fantastic.",
    "This movie is terrible. I hate it.",
    "This movie is okay. It's neither good nor bad."
]

# 分析情感
results = sentiment_analysis(texts)

# 显示结果
for text, result in zip(texts, results):
    print(f"Text: {text}")
    print(f"Sentiment: {result['label']}, Score: {result['score']:.4f}")
    print()
```

### 26.6 命名实体识别

命名实体识别（NER）是识别文本中命名实体（如人物、组织、地点等）的任务。

```python
from transformers import pipeline

# 加载命名实体识别管道
ner = pipeline('ner', grouped_entities=True)
```

### 26.7 文本摘要

文本摘要是生成文本摘要的任务，包括抽取式摘要和生成式摘要。

```python
from transformers import pipeline

# 加载文本摘要管道
summarizer = pipeline('summarization')

# 测试文本
text = """Python is an interpreted, high-level, general-purpose programming language. Created by Guido van Rossum and first released in 1991, Python's design philosophy emphasizes code readability with its notable use of significant whitespace. Its language constructs and object-oriented approach aim to help programmers write clear, logical code for small and large-scale projects. Python is dynamically typed and garbage-collected. It supports multiple programming paradigms, including structured (particularly procedural), object-oriented and functional programming. Python is often described as a "batteries included" language due to its comprehensive standard library."""

# 生成摘要
summary = summarizer(text, max_length=100, min_length=30, do_sample=False)

# 显示结果
print("Original text:")
print(text)
print("\nSummary:")
print(summary[0]['summary_text'])
```

### 26.9 问答系统

问答系统是回答用户问题的系统，包括抽取式问答和生成式问答。

```python
from transformers import pipeline

# 加载问答管道
qa_pipeline = pipeline('question-answering')

# 上下文
context = """Python is an interpreted, high-level, general-purpose programming language. Created by Guido van Rossum and first released in 1991, Python's design philosophy emphasizes code readability with its notable use of significant whitespace. Its language constructs and object-oriented approach aim to help programmers write clear, logical code for small and large-scale projects. Python is dynamically typed and garbage-collected. It supports multiple programming paradigms, including structured (particularly procedural), object-oriented and functional programming. Python is often described as a "batteries included" language due to its comprehensive standard library."""

# 问题
questions = [
    "Who created Python?",
    "When was Python first released?",
    "What is Python's design philosophy?",
    "What programming paradigms does Python support?"
]

# 回答问题
for question in questions:
    result = qa_pipeline(question=question, context=context)
    print(f"Question: {question}")
    print(f"Answer: {result['answer']}")
    print(f"Score: {result['score']:.4f}")
    print()
```

### 26.12 自然语言处理的挑战

- **歧义性**：自然语言存在大量的歧义
- **多语言性**：不同语言有不同的语法和语义
- **上下文依赖**：词语的含义依赖于上下文
- **数据稀疏性**：某些语言或领域的数据有限
- **计算资源**：深度学习模型需要大量的计算资源
- **可解释性**：模型决策的可解释性
- **伦理问题**：偏见、隐私、滥用等问题
