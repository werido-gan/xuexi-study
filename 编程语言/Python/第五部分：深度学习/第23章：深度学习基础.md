### 23.1 深度学习简介

深度学习是机器学习的一个分支，它使用多层神经网络来学习数据中的复杂模式。

> **深度学习的优势**：
> - 能够自动学习特征
> - 处理复杂的非线性关系
> - 在大规模数据上表现优异
> - 适用于图像、音频、文本等复杂数据

### 23.2 神经网络基础

#### 23.2.1 感知器

感知器是神经网络的基本单元，它接收输入，计算加权和，然后通过激活函数输出结果。

```python
import numpy as np

# 简单感知器实现
class Perceptron:
    def __init__(self, input_size):
        self.weights = np.random.randn(input_size)
        self.bias = np.random.randn()
    
    def activate(self, x):
        # 线性加权和
        z = np.dot(x, self.weights) + self.bias
        # 阶跃函数
        return 1 if z > 0 else 0
    
    def train(self, X, y, epochs=100, learning_rate=0.1):
        for _ in range(epochs):
            for i in range(len(X)):
                prediction = self.activate(X[i])
                error = y[i] - prediction
                # 更新权重和偏置
                self.weights += learning_rate * error * X[i]
                self.bias += learning_rate * error

# 测试感知器
X = np.array([[0, 0], [0, 1], [1, 0], [1, 1]])
y = np.array([0, 0, 0, 1])  # 与门

perceptron = Perceptron(input_size=2)
perceptron.train(X, y)

# 测试
for i in range(len(X)):
    print(f"Input: {X[i]}, Output: {perceptron.activate(X[i])}")
```

#### 23.2.2 多层神经网络

多层神经网络由输入层、隐藏层和输出层组成。

```
输入层 → 隐藏层1 → 隐藏层2 → ... → 输出层
```

#### 23.2.3 激活函数

激活函数为神经网络引入非线性，使网络能够学习复杂的函数。

- **Sigmoid**：将输出映射到 0-1 之间
- **Tanh**：将输出映射到 -1-1 之间
- **ReLU**：线性整流函数，x > 0 时输出 x，否则输出 0
- **Leaky ReLU**：ReLU 的变体，x < 0 时输出小的负值
- **Softmax**：用于多分类问题，将输出转换为概率分布

```python
import numpy as np
import matplotlib.pyplot as plt

# 激活函数
x = np.linspace(-5, 5, 100)

# Sigmoid
sigmoid = 1 / (1 + np.exp(-x))

# Tanh
tanh = np.tanh(x)

# ReLU
relu = np.maximum(0, x)

# Leaky ReLU
leaky_relu = np.where(x > 0, x, 0.01 * x)

# Softmax
softmax = np.exp(x) / np.sum(np.exp(x))

# 可视化
plt.figure(figsize=(12, 8))
plt.subplot(231)
plt.plot(x, sigmoid)
plt.title('Sigmoid')

plt.subplot(232)
plt.plot(x, tanh)
plt.title('Tanh')

plt.subplot(233)
plt.plot(x, relu)
plt.title('ReLU')

plt.subplot(234)
plt.plot(x, leaky_relu)
plt.title('Leaky ReLU')

plt.subplot(235)
plt.plot(x, softmax)
plt.title('Softmax')

plt.tight_layout()
plt.show()
```

### 23.3 前向传播

前向传播是指数据从输入层流向输出层的过程。

```python
import numpy as np

# 简单的前向传播
class NeuralNetwork:
    def __init__(self):
        # 初始化权重
        self.w1 = np.random.randn(2, 3)  # 输入层到隐藏层
        self.b1 = np.random.randn(3)     # 隐藏层偏置
        self.w2 = np.random.randn(3, 1)  # 隐藏层到输出层
        self.b2 = np.random.randn(1)     # 输出层偏置
    
    def relu(self, x):
        return np.maximum(0, x)
    
    def forward(self, x):
        # 输入层到隐藏层
        z1 = np.dot(x, self.w1) + self.b1
        a1 = self.relu(z1)
        
        # 隐藏层到输出层
        z2 = np.dot(a1, self.w2) + self.b2
        a2 = self.relu(z2)
        
        return a2

# 测试
nn = NeuralNetwork()
X = np.array([[0, 0], [0, 1], [1, 0], [1, 1]])
for x in X:
    print(f"Input: {x}, Output: {nn.forward(x)}")
```

### 23.4 反向传播

反向传播是指误差从输出层反向传播到输入层的过程，用于更新权重。

```python
import numpy as np

class NeuralNetwork:
    def __init__(self):
        # 初始化权重
        self.w1 = np.random.randn(2, 3)  # 输入层到隐藏层
        self.b1 = np.random.randn(3)     # 隐藏层偏置
        self.w2 = np.random.randn(3, 1)  # 隐藏层到输出层
        self.b2 = np.random.randn(1)     # 输出层偏置
    
    def relu(self, x):
        return np.maximum(0, x)
    
    def relu_derivative(self, x):
        return np.where(x > 0, 1, 0)
    
    def forward(self, x):
        # 输入层到隐藏层
        z1 = np.dot(x, self.w1) + self.b1
        a1 = self.relu(z1)
        
        # 隐藏层到输出层
        z2 = np.dot(a1, self.w2) + self.b2
        a2 = self.relu(z2)
        
        return a2, a1, z1, z2
    
    def backward(self, x, y, a2, a1, z1, z2, learning_rate=0.1):
        # 计算输出层误差
        error_output = y - a2
        delta_output = error_output * self.relu_derivative(z2)
        
        # 计算隐藏层误差
        error_hidden = np.dot(delta_output, self.w2.T)
        delta_hidden = error_hidden * self.relu_derivative(z1)
        
        # 更新权重和偏置
        self.w2 += learning_rate * np.dot(a1.T, delta_output)
        self.b2 += learning_rate * np.sum(delta_output, axis=0)
        self.w1 += learning_rate * np.dot(x.T, delta_hidden)
        self.b1 += learning_rate * np.sum(delta_hidden, axis=0)
    
    def train(self, X, y, epochs=1000, learning_rate=0.1):
        for epoch in range(epochs):
            for i in range(len(X)):
                x = X[i].reshape(1, -1)
                target = y[i].reshape(1, -1)
                a2, a1, z1, z2 = self.forward(x)
                self.backward(x, target, a2, a1, z1, z2, learning_rate)
            
            if epoch % 100 == 0:
                loss = np.mean((y - self.predict(X)) ** 2)
                print(f"Epoch {epoch}, Loss: {loss}")
    
    def predict(self, X):
        predictions = []
        for x in X:
            a2, _, _, _ = self.forward(x.reshape(1, -1))
            predictions.append(a2[0][0])
        return np.array(predictions)

# 测试
X = np.array([[0, 0], [0, 1], [1, 0], [1, 1]])
y = np.array([0, 1, 1, 0])  # 异或门

nn = NeuralNetwork()
nn.train(X, y, epochs=10000, learning_rate=0.1)

# 测试
for i in range(len(X)):
    print(f"Input: {X[i]}, Target: {y[i]}, Prediction: {nn.predict(X)[i]}")
```

### 23.5 损失函数

损失函数用于衡量模型预测与真实值之间的差异。

- **均方误差（MSE）**：适用于回归问题
- **交叉熵损失**：适用于分类问题
- **二元交叉熵**：适用于二分类问题

```python
import numpy as np

# 均方误差
def mse(y_true, y_pred):
    return np.mean((y_true - y_pred) ** 2)

# 交叉熵损失
def cross_entropy(y_true, y_pred):
    epsilon = 1e-15
    y_pred = np.clip(y_pred, epsilon, 1 - epsilon)
    return -np.mean(y_true * np.log(y_pred) + (1 - y_true) * np.log(1 - y_pred))

# 测试
y_true = np.array([0, 1, 1, 0])
y_pred = np.array([0.1, 0.9, 0.8, 0.2])

print(f"MSE: {mse(y_true, y_pred)}")
print(f"Cross Entropy: {cross_entropy(y_true, y_pred)}")
```

### 23.6 优化器

优化器用于更新模型的权重，以最小化损失函数。

- **随机梯度下降（SGD）**：基本的优化算法
- **动量优化**：考虑之前的梯度
- **Adam**：结合了动量和自适应学习率
- **RMSprop**：自适应学习率的优化算法

```python
# SGD 优化器
class SGD:
    def __init__(self, learning_rate=0.01):
        self.learning_rate = learning_rate
    
    def update(self, params, grads):
        for param, grad in zip(params, grads):
            param -= self.learning_rate * grad

# 动量优化器
class Momentum:
    def __init__(self, learning_rate=0.01, momentum=0.9):
        self.learning_rate = learning_rate
        self.momentum = momentum
        self.velocities = None
    
    def update(self, params, grads):
        if self.velocities is None:
            self.velocities = [np.zeros_like(p) for p in params]
        
        for i, (param, grad) in enumerate(zip(params, grads)):
            self.velocities[i] = self.momentum * self.velocities[i] - self.learning_rate * grad
            param += self.velocities[i]

# Adam 优化器
class Adam:
    def __init__(self, learning_rate=0.001, beta1=0.9, beta2=0.999, epsilon=1e-8):
        self.learning_rate = learning_rate
        self.beta1 = beta1
        self.beta2 = beta2
        self.epsilon = epsilon
        self.m = None
        self.v = None
        self.t = 0
    
    def update(self, params, grads):
        if self.m is None:
            self.m = [np.zeros_like(p) for p in params]
            self.v = [np.zeros_like(p) for p in params]
        
        self.t += 1
        
        for i, (param, grad) in enumerate(zip(params, grads)):
            self.m[i] = self.beta1 * self.m[i] + (1 - self.beta1) * grad
            self.v[i] = self.beta2 * self.v[i] + (1 - self.beta2) * grad ** 2
            
            m_hat = self.m[i] / (1 - self.beta1 ** self.t)
            v_hat = self.v[i] / (1 - self.beta2 ** self.t)
            
            param -= self.learning_rate * m_hat / (np.sqrt(v_hat) + self.epsilon)
```

### 23.7 深度学习框架

- **PyTorch**：Facebook 开发的深度学习框架，动态计算图

### 23.8 深度学习的挑战

- **计算资源**：需要大量的计算资源
- **数据需求**：需要大量的标记数据
- **过拟合**：容易过拟合训练数据
- **可解释性**：模型决策过程难以解释
- **调参困难**：需要调整大量超参数

### 23.9 深度学习的应用

1. **计算机视觉**：图像分类、目标检测、图像分割
2. **自然语言处理**：文本分类、机器翻译、情感分析
3. **语音识别**：语音转文本、说话人识别

### 23.10 深度学习的发展趋势

- **模型压缩**：使模型更轻量，适合部署在边缘设备
- **自监督学习**：减少对标记数据的依赖
- **联邦学习**：在保护数据隐私的前提下进行模型训练
- **小样本学习**：从少量数据中学习
- **多模态学习**：融合多种类型的数据
- **可解释性 AI**：提高模型的可解释性
- **量子机器学习**：结合量子计算和机器学习
