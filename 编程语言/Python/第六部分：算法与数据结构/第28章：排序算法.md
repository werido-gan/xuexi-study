## 第28章：排序算法 🔄

### 28.1 排序算法简介

排序算法是将一组数据按照特定顺序排列的算法。

> **排序算法的重要性**：
> - 提高数据查找效率
> - 便于数据管理和分析
> - 是许多其他算法的基础

### 28.2 常见排序算法

#### 28.2.1 冒泡排序（Bubble Sort）

冒泡排序是一种简单的排序算法，它重复地遍历要排序的数组，比较相邻的两个元素，如果它们的顺序错误就交换它们。

```python
def bubble_sort(arr):
    n = len(arr)
    for i in range(n):
        # 标记是否发生交换
        swapped = False
        for j in range(0, n - i - 1):
            if arr[j] > arr[j + 1]:
                arr[j], arr[j + 1] = arr[j + 1], arr[j]
                swapped = True
        # 如果没有交换，说明已经有序
        if not swapped:
            break
    return arr

# 测试
arr = [64, 34, 25, 12, 22, 11, 90]
print(f"Original array: {arr}")
sorted_arr = bubble_sort(arr)
print(f"Sorted array: {sorted_arr}")
```

**时间复杂度**：
- 最好情况：O(n)（已排序）
- 最坏情况：O(n²)（逆序）
- 平均情况：O(n²)

**空间复杂度**：O(1)

#### 28.2.2 选择排序（Selection Sort）

选择排序是一种简单的排序算法，它每次从待排序的数据中选出最小（或最大）的一个元素，放到序列的起始位置。

```python
def selection_sort(arr):
    n = len(arr)
    for i in range(n):
        # 找到最小元素的索引
        min_idx = i
        for j in range(i + 1, n):
            if arr[j] < arr[min_idx]:
                min_idx = j
        # 交换元素
        arr[i], arr[min_idx] = arr[min_idx], arr[i]
    return arr

# 测试
arr = [64, 34, 25, 12, 22, 11, 90]
print(f"Original array: {arr}")
sorted_arr = selection_sort(arr)
print(f"Sorted array: {sorted_arr}")
```

**时间复杂度**：
- 最好情况：O(n²)
- 最坏情况：O(n²)
- 平均情况：O(n²)

**空间复杂度**：O(1)

#### 28.2.3 插入排序（Insertion Sort）

插入排序是一种简单的排序算法，它将数组分为已排序和未排序两部分，每次将未排序部分的第一个元素插入到已排序部分的适当位置。

```python
def insertion_sort(arr):
    n = len(arr)
    for i in range(1, n):
        key = arr[i]
        j = i - 1
        # 将大于key的元素向右移动
        while j >= 0 and arr[j] > key:
            arr[j + 1] = arr[j]
            j -= 1
        # 插入key
        arr[j + 1] = key
    return arr

# 测试
arr = [64, 34, 25, 12, 22, 11, 90]
print(f"Original array: {arr}")
sorted_arr = insertion_sort(arr)
print(f"Sorted array: {sorted_arr}")
```

**时间复杂度**：
- 最好情况：O(n)（已排序）
- 最坏情况：O(n²)（逆序）
- 平均情况：O(n²)

**空间复杂度**：O(1)

#### 28.2.4 希尔排序（Shell Sort）

希尔排序是插入排序的改进版，它通过比较相距一定间隔的元素来工作，逐步减小间隔直到只比较相邻元素。

```python
def shell_sort(arr):
    n = len(arr)
    gap = n // 2
    while gap > 0:
        for i in range(gap, n):
            key = arr[i]
            j = i
            while j >= gap and arr[j - gap] > key:
                arr[j] = arr[j - gap]
                j -= gap
            arr[j] = key
        gap //= 2
    return arr

# 测试
arr = [64, 34, 25, 12, 22, 11, 90]
print(f"Original array: {arr}")
sorted_arr = shell_sort(arr)
print(f"Sorted array: {sorted_arr}")
```

**时间复杂度**：
- 最好情况：O(n log n)
- 最坏情况：O(n²)
- 平均情况：O(n log n)

**空间复杂度**：O(1)

#### 28.2.5 归并排序（Merge Sort）

归并排序是一种分治算法，它将数组分成两半，分别排序，然后将排序好的两半合并。

```python
def merge_sort(arr):
    if len(arr) <= 1:
        return arr
    # 分割数组
    mid = len(arr) // 2
    left = merge_sort(arr[:mid])
    right = merge_sort(arr[mid:])
    # 合并数组
    return merge(left, right)

def merge(left, right):
    result = []
    i = j = 0
    while i < len(left) and j < len(right):
        if left[i] < right[j]:
            result.append(left[i])
            i += 1
        else:
            result.append(right[j])
            j += 1
    # 添加剩余元素
    result.extend(left[i:])
    result.extend(right[j:])
    return result

# 测试
arr = [64, 34, 25, 12, 22, 11, 90]
print(f"Original array: {arr}")
sorted_arr = merge_sort(arr)
print(f"Sorted array: {sorted_arr}")
```

**时间复杂度**：
- 最好情况：O(n log n)
- 最坏情况：O(n log n)
- 平均情况：O(n log n)

**空间复杂度**：O(n)

#### 28.2.6 快速排序（Quick Sort）

快速排序是一种分治算法，它选择一个基准元素，将数组分为小于基准和大于基准的两部分，然后递归地排序这两部分。

```python
def quick_sort(arr):
    if len(arr) <= 1:
        return arr
    # 选择基准元素
    pivot = arr[len(arr) // 2]
    # 分割数组
    left = [x for x in arr if x < pivot]
    middle = [x for x in arr if x == pivot]
    right = [x for x in arr if x > pivot]
    # 递归排序
    return quick_sort(left) + middle + quick_sort(right)

# 测试
arr = [64, 34, 25, 12, 22, 11, 90]
print(f"Original array: {arr}")
sorted_arr = quick_sort(arr)
print(f"Sorted array: {sorted_arr}")

# 原地快速排序
def quick_sort_in_place(arr, low, high):
    if low < high:
        # 分区
        pi = partition(arr, low, high)
        # 递归排序左右两部分
        quick_sort_in_place(arr, low, pi - 1)
        quick_sort_in_place(arr, pi + 1, high)

def partition(arr, low, high):
    pivot = arr[high]
    i = low - 1
    for j in range(low, high):
        if arr[j] <= pivot:
            i += 1
            arr[i], arr[j] = arr[j], arr[i]
    arr[i + 1], arr[high] = arr[high], arr[i + 1]
    return i + 1

# 测试原地快速排序
arr = [64, 34, 25, 12, 22, 11, 90]
print(f"Original array: {arr}")
quick_sort_in_place(arr, 0, len(arr) - 1)
print(f"Sorted array: {arr}")
```

**时间复杂度**：
- 最好情况：O(n log n)
- 最坏情况：O(n²)（已排序或逆序）
- 平均情况：O(n log n)

**空间复杂度**：O(log n)（递归栈）

#### 28.2.7 堆排序（Heap Sort）

堆排序是一种基于堆数据结构的排序算法，它利用堆的性质来排序数组。

```python
def heap_sort(arr):
    n = len(arr)
    # 构建最大堆
    for i in range(n // 2 - 1, -1, -1):
        heapify(arr, n, i)
    # 逐个提取元素
    for i in range(n - 1, 0, -1):
        arr[i], arr[0] = arr[0], arr[i]  # 交换
        heapify(arr, i, 0)  # 调整堆
    return arr

def heapify(arr, n, i):
    largest = i  # 初始化最大值为根节点
    left = 2 * i + 1  # 左子节点
    right = 2 * i + 2  # 右子节点
    # 如果左子节点大于根节点
    if left < n and arr[left] > arr[largest]:
        largest = left
    # 如果右子节点大于当前最大值
    if right < n and arr[right] > arr[largest]:
        largest = right
    # 如果最大值不是根节点
    if largest != i:
        arr[i], arr[largest] = arr[largest], arr[i]  # 交换
        heapify(arr, n, largest)  # 递归调整

# 测试
arr = [64, 34, 25, 12, 22, 11, 90]
print(f"Original array: {arr}")
sorted_arr = heap_sort(arr)
print(f"Sorted array: {sorted_arr}")
```

**时间复杂度**：
- 最好情况：O(n log n)
- 最坏情况：O(n log n)
- 平均情况：O(n log n)

**空间复杂度**：O(1)

#### 28.2.8 计数排序（Counting Sort）

计数排序是一种非比较排序算法，它通过计数来排序数组。

```python
def counting_sort(arr):
    if not arr:
        return arr
    # 找到最大值和最小值
    max_val = max(arr)
    min_val = min(arr)
    # 创建计数数组
    count = [0] * (max_val - min_val + 1)
    # 计数
    for num in arr:
        count[num - min_val] += 1
    # 重建数组
    sorted_arr = []
    for i in range(len(count)):
        sorted_arr.extend([i + min_val] * count[i])
    return sorted_arr

# 测试
arr = [64, 34, 25, 12, 22, 11, 90]
print(f"Original array: {arr}")
sorted_arr = counting_sort(arr)
print(f"Sorted array: {sorted_arr}")
```

**时间复杂度**：O(n + k)，其中 k 是数组元素的范围

**空间复杂度**：O(k)

#### 28.2.9 桶排序（Bucket Sort）

桶排序是一种非比较排序算法，它将数组元素分配到不同的桶中，然后对每个桶进行排序。

```python
def bucket_sort(arr):
    if not arr:
        return arr
    # 找到最大值和最小值
    max_val = max(arr)
    min_val = min(arr)
    # 计算桶的数量
    bucket_count = len(arr)
    # 创建桶
    buckets = [[] for _ in range(bucket_count)]
    # 将元素分配到桶中
    for num in arr:
        index = int((num - min_val) * (bucket_count - 1) / (max_val - min_val))
        buckets[index].append(num)
    # 对每个桶排序
    for i in range(bucket_count):
        buckets[i].sort()
    # 合并桶
    sorted_arr = []
    for bucket in buckets:
        sorted_arr.extend(bucket)
    return sorted_arr

# 测试
arr = [64, 34, 25, 12, 22, 11, 90]
print(f"Original array: {arr}")
sorted_arr = bucket_sort(arr)
print(f"Sorted array: {sorted_arr}")
```

**时间复杂度**：
- 最好情况：O(n + k)
- 最坏情况：O(n²)
- 平均情况：O(n + k)

**空间复杂度**：O(n + k)

#### 28.2.10 基数排序（Radix Sort）

基数排序是一种非比较排序算法，它按照位数从低到高进行排序。

```python
def radix_sort(arr):
    if not arr:
        return arr
    # 找到最大值
    max_val = max(arr)
    # 计算最大位数
    max_digits = len(str(max_val))
    # 从低位到高位排序
    for i in range(max_digits):
        # 创建桶
        buckets = [[] for _ in range(10)]
        # 将元素分配到桶中
        for num in arr:
            digit = (num // 10 ** i) % 10
            buckets[digit].append(num)
        # 合并桶
        arr = []
        for bucket in buckets:
            arr.extend(bucket)
    return arr

# 测试
arr = [64, 34, 25, 12, 22, 11, 90]
print(f"Original array: {arr}")
sorted_arr = radix_sort(arr)
print(f"Sorted array: {sorted_arr}")
```

**时间复杂度**：O(n * k)，其中 k 是最大数的位数

**空间复杂度**：O(n + k)

### 28.3 排序算法的比较

| 排序算法 | 时间复杂度（最好） | 时间复杂度（最坏） | 时间复杂度（平均） | 空间复杂度 | 稳定性 |
|---------|------------------|------------------|------------------|-----------|--------|
| 冒泡排序 | O(n) | O(n²) | O(n²) | O(1) | 稳定 |
| 选择排序 | O(n²) | O(n²) | O(n²) | O(1) | 不稳定 |
| 插入排序 | O(n) | O(n²) | O(n²) | O(1) | 稳定 |
| 希尔排序 | O(n log n) | O(n²) | O(n log n) | O(1) | 不稳定 |
| 归并排序 | O(n log n) | O(n log n) | O(n log n) | O(n) | 稳定 |
| 快速排序 | O(n log n) | O(n²) | O(n log n) | O(log n) | 不稳定 |
| 堆排序 | O(n log n) | O(n log n) | O(n log n) | O(1) | 不稳定 |
| 计数排序 | O(n + k) | O(n + k) | O(n + k) | O(k) | 稳定 |
| 桶排序 | O(n + k) | O(n²) | O(n + k) | O(n + k) | 稳定 |
| 基数排序 | O(n * k) | O(n * k) | O(n * k) | O(n + k) | 稳定 |

### 28.4 排序算法的选择

1. **数据规模小**：插入排序、选择排序、冒泡排序
2. **数据基本有序**：插入排序、冒泡排序
3. **数据规模大**：快速排序、归并排序、堆排序
4. **数据范围小**：计数排序、桶排序、基数排序
5. **需要稳定排序**：归并排序、冒泡排序、插入排序、计数排序、桶排序、基数排序
6. **空间限制**：堆排序、希尔排序、插入排序、选择排序、冒泡排序

### 28.5 Python 内置排序函数

Python 提供了内置的排序函数 `sorted()` 和列表方法 `sort()`。


### 28.7 排序算法的优化

1. **冒泡排序优化**：添加交换标记，提前结束排序
2. **快速排序优化**：
   - 选择合适的基准元素（如随机选择、三数取中法）
   - 对于小规模数据使用插入排序
   - 递归深度限制，防止栈溢出
3. **归并排序优化**：
   - 对于小规模数据使用插入排序
   - 原地归并，减少空间使用
4. **堆排序优化**：使用 Floyd 建堆法，减少建堆时间
5. **计数排序优化**：处理负整数，使用偏移量
6. **桶排序优化**：根据数据分布调整桶的数量和大小

### 28.9 排序算法的性能测试

```python
import time
import random

# 生成测试数据
def generate_data(size):
    return [random.randint(0, 100000) for _ in range(size)]

def test_sort_algorithm(sort_func, data):
    start_time = time.time()
    sorted_data = sort_func(data.copy())

# 测试不同排序算法
```
