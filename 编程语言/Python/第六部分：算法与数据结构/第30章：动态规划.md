## 第30章：动态规划 📊

### 30.1 动态规划简介

动态规划（Dynamic Programming，简称 DP）是一种解决复杂问题的算法思想，它通过将问题分解为子问题并存储子问题的解来避免重复计算。

> **动态规划的核心思想**：
> - 最优子结构：问题的最优解包含子问题的最优解
> - 重叠子问题：不同的子问题可能包含相同的子子问题
> - 记忆化：存储子问题的解以避免重复计算

### 30.2 动态规划的基本步骤

1. **定义状态**：确定问题的状态表示
2. **确定状态转移方程**：描述状态之间的转移关系
3. **初始化**：设置初始状态的值
4. **计算顺序**：按照依赖关系计算所有状态
5. **返回结果**：根据最终状态得到问题的解

### 30.3 经典动态规划问题

#### 30.3.1 斐波那契数列

**问题**：计算斐波那契数列的第 n 项。

**状态定义**：`dp[i]` 表示第 i 个斐波那契数。

**状态转移方程**：`dp[i] = dp[i-1] + dp[i-2]`。

**初始化**：`dp[0] = 0, dp[1] = 1`。

```python
def fibonacci(n):
    """计算斐波那契数列的第 n 项"""
    if n <= 1:
        return n
    dp = [0] * (n + 1)
    dp[0] = 0
    dp[1] = 1
    for i in range(2, n + 1):
        dp[i] = dp[i-1] + dp[i-2]
    return dp[n]

# 测试
n = 10
print(f"Fibonacci({n}): {fibonacci(n)}")  # 55

# 优化空间复杂度
def fibonacci_optimized(n):
    """计算斐波那契数列的第 n 项（空间优化）"""
    if n <= 1:
        return n
    a, b = 0, 1
    for _ in range(2, n + 1):
        a, b = b, a + b
    return b

# 测试
n = 10
print(f"Fibonacci({n}) (optimized): {fibonacci_optimized(n)}")  # 55
```

#### 30.3.2 爬楼梯

**问题**：一个人爬楼梯，每次可以爬 1 或 2 个台阶，求爬到第 n 个台阶的方法数。

**状态定义**：`dp[i]` 表示爬到第 i 个台阶的方法数。

**状态转移方程**：`dp[i] = dp[i-1] + dp[i-2]`。

**初始化**：`dp[1] = 1, dp[2] = 2`。

```python
def climb_stairs(n):
    """计算爬到第 n 个台阶的方法数"""
    if n <= 2:
        return n
    dp = [0] * (n + 1)
    dp[1] = 1
    dp[2] = 2
    for i in range(3, n + 1):
        dp[i] = dp[i-1] + dp[i-2]
    return dp[n]

# 测试

        a, b = b, a + b
    return b
```

#### 30.3.3 最大子数组和

**问题**：给定一个整数数组，找到一个具有最大和的连续子数组，返回其最大和。

**状态定义**：`dp[i]` 表示以第 i 个元素结尾的最大子数组和。

**状态转移方程**：`dp[i] = max(nums[i], dp[i-1] + nums[i])`。

**初始化**：`dp[0] = nums[0]`。

```python
def max_subarray(nums):
    """计算最大子数组和"""
    if not nums:
        return 0
    n = len(nums)
    dp = [0] * n
    dp[0] = nums[0]
    max_sum = dp[0]
    for i in range(1, n):
        dp[i] = max(nums[i], dp[i-1] + nums[i])
        max_sum = max(max_sum, dp[i])
    return max_sum

# 测试
nums = [-2, 1, -3, 4, -1, 2, 1, -5, 4]
print(f"Max subarray sum: {max_subarray(nums)}")  # 6

# 优化空间复杂度
def max_subarray_optimized(nums):
    """计算最大子数组和（空间优化）"""
    if not nums:
        return 0
    current_sum = max_sum = nums[0]
    for num in nums[1:]:
        current_sum = max(num, current_sum + num)
        max_sum = max(max_sum, current_sum)
    return max_sum

# 测试
nums = [-2, 1, -3, 4, -1, 2, 1, -5, 4]
print(f"Max subarray sum (optimized): {max_subarray_optimized(nums)}")  # 6
```

#### 30.3.4 最长递增子序列

**问题**：给定一个整数数组，找到最长递增子序列的长度。

**状态定义**：`dp[i]` 表示以第 i 个元素结尾的最长递增子序列的长度。

**状态转移方程**：`dp[i] = max(dp[j] + 1) for j < i if nums[j] < nums[i]`。

**初始化**：`dp[i] = 1`（每个元素自身构成一个长度为 1 的子序列）。

```python
def length_of_lis(nums):
    """计算最长递增子序列的长度"""
    if not nums:
        return 0
    n = len(nums)
    dp = [1] * n
    for i in range(1, n):
        for j in range(i):
            if nums[j] < nums[i]:
                dp[i] = max(dp[i], dp[j] + 1)
    return max(dp)

# 测试
nums = [10, 9, 2, 5, 3, 7, 101, 18]
print(f"Length of LIS: {length_of_lis(nums)}")  # 4

# 优化时间复杂度（使用二分查找）
def length_of_lis_optimized(nums):
    """计算最长递增子序列的长度（时间优化）"""
    if not nums:
        return 0
    tails = []
    for num in nums:
        # 二分查找找到第一个大于等于 num 的位置
        left, right = 0, len(tails)
        while left < right:
            mid = (left + right) // 2
            if tails[mid] < num:
                left = mid + 1
            else:
                right = mid
        # 如果找到位置，更新 tails[left]，否则添加到末尾
        if left == len(tails):
            tails.append(num)
        else:
            tails[left] = num
    return len(tails)

# 测试
nums = [10, 9, 2, 5, 3, 7, 101, 18]
print(f"Length of LIS (optimized): {length_of_lis_optimized(nums)}")  # 4
```

#### 30.3.5 0-1 背包问题

**问题**：给定一组物品，每个物品有重量和价值，在背包容量有限的情况下，选择哪些物品使得总价值最大。

**状态定义**：`dp[i][j]` 表示前 i 个物品，背包容量为 j 时的最大价值。

**状态转移方程**：`dp[i][j] = max(dp[i-1][j], dp[i-1][j-weight[i]] + value[i])`。

**初始化**：`dp[0][j] = 0, dp[i][0] = 0`。

```python
def knapsack(weights, values, capacity):
    """0-1 背包问题"""
    n = len(weights)
    # 创建二维数组
    dp = [[0] * (capacity + 1) for _ in range(n + 1)]
    for i in range(1, n + 1):
        for j in range(1, capacity + 1):
            if weights[i-1] <= j:
                # 选择第 i 个物品或不选择
                dp[i][j] = max(dp[i-1][j], dp[i-1][j-weights[i-1]] + values[i-1])
            else:
                # 不选择第 i 个物品
                dp[i][j] = dp[i-1][j]
    return dp[n][capacity]

# 测试
weights = [2, 3, 4, 5]
values = [3, 4, 5, 6]
capacity = 8
print(f"Max value: {knapsack(weights, values, capacity)}")  # 10

# 优化空间复杂度
def knapsack_optimized(weights, values, capacity):
    """0-1 背包问题（空间优化）"""
    n = len(weights)
    # 创建一维数组
    dp = [0] * (capacity + 1)
    for i in range(n):
        # 逆序遍历，避免重复选择
        for j in range(capacity, weights[i]-1, -1):
            dp[j] = max(dp[j], dp[j-weights[i]] + values[i])
    return dp[capacity]

# 测试
weights = [2, 3, 4, 5]
values = [3, 4, 5, 6]
capacity = 8
print(f"Max value (optimized): {knapsack_optimized(weights, values, capacity)}")  # 10
```

#### 30.3.6 完全背包问题

**问题**：给定一组物品，每个物品可以选多次，在背包容量有限的情况下，选择哪些物品使得总价值最大。

**状态定义**：`dp[j]` 表示背包容量为 j 时的最大价值。

**状态转移方程**：`dp[j] = max(dp[j], dp[j-weight[i]] + value[i])`。

**初始化**：`dp[0] = 0`。

```python
def unbounded_knapsack(weights, values, capacity):
    """完全背包问题"""
    n = len(weights)
    dp = [0] * (capacity + 1)
    for i in range(n):
        # 正序遍历，允许重复选择
        for j in range(weights[i], capacity + 1):
            dp[j] = max(dp[j], dp[j-weights[i]] + values[i])
    return dp[capacity]

# 测试
weights = [2, 3, 4, 5]
values = [3, 4, 5, 6]
capacity = 8
print(f"Max value: {unbounded_knapsack(weights, values, capacity)}")  # 12
```

#### 30.3.7 最长公共子序列

**问题**：给定两个字符串，找到它们的最长公共子序列的长度。

**状态定义**：`dp[i][j]` 表示字符串 s1 的前 i 个字符和字符串 s2 的前 j 个字符的最长公共子序列长度。

**状态转移方程**：
- 如果 s1[i-1] == s2[j-1]，则 `dp[i][j] = dp[i-1][j-1] + 1`
- 否则，`dp[i][j] = max(dp[i-1][j], dp[i][j-1])`

**初始化**：`dp[i][0] = 0, dp[0][j] = 0`。

```python
def longest_common_subsequence(s1, s2):
    """计算最长公共子序列的长度"""
    m, n = len(s1), len(s2)
    dp = [[0] * (n + 1) for _ in range(m + 1)]
    for i in range(1, m + 1):
        for j in range(1, n + 1):
            if s1[i-1] == s2[j-1]:
                dp[i][j] = dp[i-1][j-1] + 1
            else:
                dp[i][j] = max(dp[i-1][j], dp[i][j-1])
    return dp[m][n]

# 测试
s1 = "abcde"
s2 = "ace"
print(f"Length of LCS: {longest_common_subsequence(s1, s2)}")  # 3

# 优化空间复杂度
def longest_common_subsequence_optimized(s1, s2):
    """计算最长公共子序列的长度（空间优化）"""
    m, n = len(s1), len(s2)
    # 确保 s2 是较短的字符串
    if m < n:
        s1, s2 = s2, s1
        m, n = n, m
    dp_prev = [0] * (n + 1)
    dp_curr = [0] * (n + 1)
    for i in range(1, m + 1):
        for j in range(1, n + 1):
            if s1[i-1] == s2[j-1]:
                dp_curr[j] = dp_prev[j-1] + 1
            else:
                dp_curr[j] = max(dp_prev[j], dp_curr[j-1])
        dp_prev, dp_curr = dp_curr, dp_prev
    return dp_prev[n]

# 测试
s1 = "abcde"
s2 = "ace"
print(f"Length of LCS (optimized): {longest_common_subsequence_optimized(s1, s2)}")  # 3
```

#### 30.3.8 编辑距离

**问题**：给定两个字符串，计算将一个字符串转换为另一个字符串所需的最少操作数，操作包括插入、删除和替换。

**状态定义**：`dp[i][j]` 表示将字符串 s1 的前 i 个字符转换为字符串 s2 的前 j 个字符所需的最少操作数。

**状态转移方程**：
- 如果 s1[i-1] == s2[j-1]，则 `dp[i][j] = dp[i-1][j-1]`
- 否则，`dp[i][j] = min(dp[i-1][j], dp[i][j-1], dp[i-1][j-1]) + 1`

**初始化**：`dp[i][0] = i, dp[0][j] = j`。

```python
def edit_distance(s1, s2):
    """计算编辑距离"""
    m, n = len(s1), len(s2)
    dp = [[0] * (n + 1) for _ in range(m + 1)]
    # 初始化
    for i in range(m + 1):
        dp[i][0] = i
    for j in range(n + 1):
        dp[0][j] = j
    # 填充 dp 表
    for i in range(1, m + 1):
        for j in range(1, n + 1):
            if s1[i-1] == s2[j-1]:
                dp[i][j] = dp[i-1][j-1]
            else:
                dp[i][j] = min(dp[i-1][j], dp[i][j-1], dp[i-1][j-1]) + 1
    return dp[m][n]

# 测试
s1 = "intention"
s2 = "execution"
print(f"Edit distance: {edit_distance(s1, s2)}")  # 5

# 优化空间复杂度
def edit_distance_optimized(s1, s2):
    """计算编辑距离（空间优化）"""
    m, n = len(s1), len(s2)
    # 确保 s2 是较短的字符串
    if m < n:
        s1, s2 = s2, s1
        m, n = n, m
    dp_prev = list(range(n + 1))
    for i in range(1, m + 1):
        dp_curr = [i] * (n + 1)
        for j in range(1, n + 1):
            if s1[i-1] == s2[j-1]:
                dp_curr[j] = dp_prev[j-1]
            else:
                dp_curr[j] = min(dp_prev[j], dp_curr[j-1], dp_prev[j-1]) + 1
        dp_prev = dp_curr
    return dp_prev[n]

# 测试
s1 = "intention"
s2 = "execution"
print(f"Edit distance (optimized): {edit_distance_optimized(s1, s2)}")  # 5
```

### 30.4 动态规划的类型

1. **线性动态规划**：状态转移是线性的，如斐波那契数列、爬楼梯
2. **区间动态规划**：状态表示区间，如最长回文子串
3. **背包问题**：0-1 背包、完全背包、多重背包
4. **树形动态规划**：在树上进行动态规划，如二叉树的最大路径和
5. **状态压缩动态规划**：使用位运算压缩状态，如旅行商问题
6. **数位动态规划**：处理数字相关的问题，如数字计数
7. **概率动态规划**：处理概率相关的问题，如期望问题
8. **博弈论动态规划**：处理博弈相关的问题，如石子游戏

### 30.5 动态规划的优化技巧

1. **空间优化**：使用滚动数组或一维数组代替二维数组
2. **状态压缩**：使用位运算或其他方式压缩状态
3. **预处理**：预处理一些信息，减少重复计算
4. **剪枝**：在计算过程中剪枝不必要的状态
5. **记忆化搜索**：使用递归和记忆化代替迭代
6. **并行计算**：利用并行计算加速状态转移

### 30.6 动态规划的应用

1. **路径问题**：最短路径、最长路径
2. **序列问题**：最长递增子序列、最长公共子序列
3. **背包问题**：资源分配、投资组合
4. **字符串问题**：编辑距离、最长回文子串
5. **图论问题**：最短路径、最小生成树

7. **游戏问题**：博弈论、策略选择
