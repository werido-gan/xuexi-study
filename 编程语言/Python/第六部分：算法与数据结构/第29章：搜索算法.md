### 29.2 线性搜索（Linear Search）

线性搜索是一种简单的搜索算法，它从数据结构的一端开始，逐个检查每个元素，直到找到目标元素或搜索完整个数据结构。

```python
def linear_search(arr, target):
    """线性搜索"""
    for i, element in enumerate(arr):
        if element == target:
            return i  # 返回目标元素的索引
    return -1  # 未找到目标元素

# 测试
arr = [64, 34, 25, 12, 22, 11, 90]
target = 25
result = linear_search(arr, target)
print(f"Target {target} found at index: {result}")  # 2

target = 100
result = linear_search(arr, target)
print(f"Target {target} found at index: {result}")  # -1
```

**时间复杂度**：
- 最好情况：O(1)（目标元素在第一个位置）
- 最坏情况：O(n)（目标元素在最后一个位置或不存在）
- 平均情况：O(n)

**空间复杂度**：O(1)

### 29.3 二分搜索（Binary Search）

二分搜索是一种高效的搜索算法，它要求数据结构已经排序。它通过每次将搜索范围减半来查找目标元素。

```python
def binary_search(arr, target):
    """二分搜索"""
    low = 0
    high = len(arr) - 1
    while low <= high:
        mid = (low + high) // 2
        if arr[mid] == target:
            return mid  # 返回目标元素的索引
        elif arr[mid] < target:
            low = mid + 1
        else:
            high = mid - 1
    return -1  # 未找到目标元素

# 测试
arr = [11, 12, 22, 25, 34, 64, 90]  # 已排序
target = 25
result = binary_search(arr, target)
print(f"Target {target} found at index: {result}")  # 3

target = 100
result = binary_search(arr, target)
print(f"Target {target} found at index: {result}")  # -1

# 递归实现
def binary_search_recursive(arr, target, low, high):
    """递归实现二分搜索"""
    if low > high:
        return -1
    mid = (low + high) // 2
    if arr[mid] == target:
        return mid
    elif arr[mid] < target:
        return binary_search_recursive(arr, target, mid + 1, high)
    else:
        return binary_search_recursive(arr, target, low, mid - 1)

# 测试递归实现
target = 25
result = binary_search_recursive(arr, target, 0, len(arr) - 1)
print(f"Target {target} found at index (recursive): {result}")  # 3
```

**时间复杂度**：
- 最好情况：O(1)（目标元素在中间位置）
- 最坏情况：O(log n)
- 平均情况：O(log n)

**空间复杂度**：
- 迭代实现：O(1)
- 递归实现：O(log n)（递归栈）

### 29.4 插值搜索（Interpolation Search）

插值搜索是二分搜索的改进版，它根据目标元素的值与搜索范围的最小值和最大值的比例来估计目标元素的位置。

```python
def interpolation_search(arr, target):
    """插值搜索"""
    low = 0
    high = len(arr) - 1
    while low <= high and target >= arr[low] and target <= arr[high]:
        if low == high:
            if arr[low] == target:
                return low
            return -1
        # 计算插值位置
        pos = low + ((target - arr[low]) * (high - low)) // (arr[high] - arr[low])
        if arr[pos] == target:
            return pos
        elif arr[pos] < target:
            low = pos + 1
        else:
            high = pos - 1
    return -1

# 测试
arr = [11, 12, 22, 25, 34, 64, 90]  # 已排序
target = 25
result = interpolation_search(arr, target)
print(f"Target {target} found at index: {result}")  # 3

target = 100
result = interpolation_search(arr, target)
print(f"Target {target} found at index: {result}")  # -1
```

**时间复杂度**：
- 最好情况：O(1)（目标元素在估计位置）
- 最坏情况：O(n)（数据分布不均匀）
- 平均情况：O(log log n)（数据分布均匀）

**空间复杂度**：O(1)

### 29.5 跳跃搜索（Jump Search）

跳跃搜索是一种介于线性搜索和二分搜索之间的搜索算法，它通过每次跳跃固定步长来减少搜索范围。

```python
import math

def jump_search(arr, target):
    """跳跃搜索"""
    n = len(arr)
    step = int(math.sqrt(n))  # 跳跃步长
    prev = 0
    # 找到包含目标元素的块
    while arr[min(step, n) - 1] < target:
        prev = step
        step += int(math.sqrt(n))
        if prev >= n:
            return -1
    # 在块内线性搜索
    while arr[prev] < target:
        prev += 1
        if prev == min(step, n):
            return -1
    if arr[prev] == target:
        return prev
    return -1

# 测试
arr = [11, 12, 22, 25, 34, 64, 90]  # 已排序
target = 25
result = jump_search(arr, target)
print(f"Target {target} found at index: {result}")  # 3

target = 100
result = jump_search(arr, target)
print(f"Target {target} found at index: {result}")  # -1
```

**时间复杂度**：
- 最好情况：O(√n)
- 最坏情况：O(√n)
- 平均情况：O(√n)

**空间复杂度**：O(1)

### 29.6 指数搜索（Exponential Search）

指数搜索是一种结合了二分搜索和线性搜索的搜索算法，它首先找到一个包含目标元素的范围，然后在该范围内进行二分搜索。

```python
def exponential_search(arr, target):
    """指数搜索"""
    n = len(arr)
    # 如果目标元素在第一个位置
    if arr[0] == target:
        return 0
    # 找到包含目标元素的范围
    i = 1
    while i < n and arr[i] <= target:
        i *= 2
    # 在找到的范围内进行二分搜索
    return binary_search(arr, target, i // 2, min(i, n - 1))

# 测试
arr = [11, 12, 22, 25, 34, 64, 90]  # 已排序
target = 25
result = exponential_search(arr, target)
print(f"Target {target} found at index: {result}")  # 3

target = 100
result = exponential_search(arr, target)
print(f"Target {target} found at index: {result}")  # -1
```

**时间复杂度**：
- 最好情况：O(1)（目标元素在第一个位置）
- 最坏情况：O(log n)
- 平均情况：O(log n)

**空间复杂度**：O(1)

### 29.7 哈希表搜索

哈希表搜索是一种通过哈希函数将键映射到值的搜索算法，它可以在常数时间内找到目标元素。

```python
# 使用字典实现哈希表搜索
def hash_search(hash_table, target):
    """哈希表搜索"""
    if target in hash_table:
        return hash_table[target]
    return -1

# 测试
hash_table = {11: 0, 12: 1, 22: 2, 25: 3, 34: 4, 64: 5, 90: 6}
target = 25
result = hash_search(hash_table, target)
print(f"Target {target} found at index: {result}")  # 3

target = 100
result = hash_search(hash_table, target)
print(f"Target {target} found at index: {result}")  # -1
```

**时间复杂度**：
- 最好情况：O(1)
- 最坏情况：O(n)（哈希冲突）
- 平均情况：O(1)

**空间复杂度**：O(n)

### 29.8 树搜索

#### 29.8.1 二叉搜索树搜索

二叉搜索树是一种特殊的二叉树，左子树的所有节点值小于根节点，右子树的所有节点值大于根节点。

```python
class TreeNode:
    def __init__(self, val=0, left=None, right=None):
        self.val = val
        self.left = left
        self.right = right

def search_bst(root, target):
    """二叉搜索树搜索"""
    if not root:
        return None
    if root.val == target:
        return root
    elif root.val < target:
        return search_bst(root.right, target)
    else:
        return search_bst(root.left, target)

# 构建二叉搜索树
root = TreeNode(5)
root.left = TreeNode(3)
root.right = TreeNode(7)
root.left.left = TreeNode(2)
root.left.right = TreeNode(4)
root.right.left = TreeNode(6)
root.right.right = TreeNode(8)

# 测试
target = 4
result = search_bst(root, target)
print(f"Target {target} found: {result.val if result else 'Not found'}")  # 4

target = 10
result = search_bst(root, target)
print(f"Target {target} found: {result.val if result else 'Not found'}")  # Not found
```

**时间复杂度**：
- 最好情况：O(log n)（平衡二叉搜索树）
- 最坏情况：O(n)（退化为链表）
- 平均情况：O(log n)

**空间复杂度**：
- 迭代实现：O(1)
- 递归实现：O(log n)（递归栈）

#### 29.8.2 红黑树搜索

红黑树是一种自平衡的二叉搜索树，它通过着色规则来保持树的平衡。

```python
# Python 中的红黑树实现（使用 bisect 模块模拟）
import bisect

class RedBlackTree:
    def __init__(self):
        self.keys = []
        self.values = []
    
    def insert(self, key, value):
        """插入键值对"""
        index = bisect.bisect_left(self.keys, key)
        if index < len(self.keys) and self.keys[index] == key:
            self.values[index] = value
        else:
            bisect.insort(self.keys, key)
            self.values.insert(index, value)
    
    def search(self, key):
        """搜索键"""
        index = bisect.bisect_left(self.keys, key)
        if index < len(self.keys) and self.keys[index] == key:
            return self.values[index]
        return -1

# 测试
rbt = RedBlackTree()
rbt.insert(5, 0)
rbt.insert(3, 1)
rbt.insert(7, 2)
rbt.insert(2, 3)
rbt.insert(4, 4)
rbt.insert(6, 5)
rbt.insert(8, 6)

target = 4
result = rbt.search(target)
print(f"Target {target} found: {result}")  # 4

target = 10
result = rbt.search(target)
print(f"Target {target} found: {result}")  # -1
```

**时间复杂度**：
- 最好情况：O(log n)
- 最坏情况：O(log n)
- 平均情况：O(log n)

**空间复杂度**：O(n)

### 29.9 图搜索

#### 29.9.1 深度优先搜索（DFS）

深度优先搜索是一种用于遍历或搜索树或图的算法，它从根节点开始，尽可能深地探索树或图的分支。

```python
def dfs(graph, start, target, visited=None):
    """深度优先搜索"""
    if visited is None:
        visited = set()
    visited.add(start)
    if start == target:
        return True
    for neighbor in graph[start]:
        if neighbor not in visited:
            if dfs(graph, neighbor, target, visited):
                return True
    return False

# 测试图
 graph = {
    'A': ['B', 'C'],
    'B': ['D', 'E'],
    'C': ['F'],
    'D': [],
    'E': ['F'],
    'F': []
}

target = 'F'
result = dfs(graph, 'A', target)
print(f"Target {target} found: {result}")  # True

target = 'G'
result = dfs(graph, 'A', target)
print(f"Target {target} found: {result}")  # False
```

**时间复杂度**：O(V + E)，其中 V 是顶点数，E 是边数

**空间复杂度**：O(V)（递归栈）

#### 29.9.2 广度优先搜索（BFS）

广度优先搜索是一种用于遍历或搜索树或图的算法，它从根节点开始，先访问所有相邻节点，然后再访问它们的相邻节点。

```python
from collections import deque

def bfs(graph, start, target):
    """广度优先搜索"""
    visited = set()
    queue = deque([start])
    visited.add(start)
    while queue:
        node = queue.popleft()
        if node == target:
            return True
        for neighbor in graph[node]:
            if neighbor not in visited:
                visited.add(neighbor)
                queue.append(neighbor)
    return False

# 测试图
graph = {
    'A': ['B', 'C'],
    'B': ['D', 'E'],
    'C': ['F'],
    'D': [],
    'E': ['F'],
    'F': []
}

target = 'F'
result = bfs(graph, 'A', target)
print(f"Target {target} found: {result}")  # True

target = 'G'
result = bfs(graph, 'A', target)
print(f"Target {target} found: {result}")  # False
```

**时间复杂度**：O(V + E)，其中 V 是顶点数，E 是边数

**空间复杂度**：O(V)

### 29.10 搜索算法的比较

| 搜索算法 | 时间复杂度（最好） | 时间复杂度（最坏） | 时间复杂度（平均） | 空间复杂度 | 适用场景 |
|---------|------------------|------------------|------------------|-----------|----------|
| 线性搜索 | O(1) | O(n) | O(n) | O(1) | 小规模数据，无序数据 |
| 二分搜索 | O(1) | O(log n) | O(log n) | O(1) | 有序数据 |
| 插值搜索 | O(1) | O(n) | O(log log n) | O(1) | 有序数据，均匀分布 |
| 跳跃搜索 | O(√n) | O(√n) | O(√n) | O(1) | 有序数据 |
| 指数搜索 | O(1) | O(log n) | O(log n) | O(1) | 有序数据，大规模数据 |
| 哈希表搜索 | O(1) | O(n) | O(1) | O(n) | 需要快速查找，空间充足 |
| 二叉搜索树搜索 | O(log n) | O(n) | O(log n) | O(n) | 动态数据，需要频繁插入删除 |
| 红黑树搜索 | O(log n) | O(log n) | O(log n) | O(n) | 动态数据，需要平衡 |
| 深度优先搜索 | O(V + E) | O(V + E) | O(V + E) | O(V) | 图遍历，路径查找 |
| 广度优先搜索 | O(V + E) | O(V + E) | O(V + E) | O(V) | 图遍历，最短路径 |

### 29.11 搜索算法的选择

1. **数据规模小**：线性搜索
2. **数据有序**：二分搜索、插值搜索、跳跃搜索、指数搜索
3. **数据分布均匀**：插值搜索
4. **数据动态变化**：二叉搜索树、红黑树
5. **需要快速查找**：哈希表搜索
6. **图结构**：深度优先搜索、广度优先搜索
7. **空间限制**：线性搜索、二分搜索、插值搜索、跳跃搜索、指数搜索
8. **时间限制**：哈希表搜索、二分搜索、红黑树搜索

### 29.16 搜索算法的性能测试

```python
import time
import random

# 生成测试数据
def generate_data(size):
    return [random.randint(0, 100000) for _ in range(size)]

# 测试搜索算法
def test_search_algorithm(search_func, data, target):
    start_time = time.time()
    result = search_func(data, target)
    end_time = time.time()
    return result, end_time - start_time

# 测试不同搜索算法
data_sizes = [1000, 5000, 10000]
search_functions = [
    ("Linear Search", linear_search),
    ("Binary Search", lambda arr, target: binary_search(sorted(arr), target)),
    ("Jump Search", lambda arr, target: jump_search(sorted(arr), target)),
    ("Exponential Search", lambda arr, target: exponential_search(sorted(arr), target)),
    ("Hash Table Search", lambda arr, target: hash_search({v: i for i, v in enumerate(arr)}, target))
]

for size in data_sizes:
    print(f"\nTesting with data size: {size}")
    data = generate_data(size)
    target = data[size // 2]  # 选择中间元素作为目标
    for name, func in search_functions:
        try:
            result, time_taken = test_search_algorithm(func, data, target)
            print(f"{name}: {time_taken:.6f} seconds, Found: {result != -1}")
        except Exception as e:
            print(f"{name}: Error - {e}")
```
