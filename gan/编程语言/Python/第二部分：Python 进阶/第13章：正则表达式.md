## 第13章：正则表达式 🎯

### 13.1 基础语法

```python
import re

# 基本匹配
pattern = r"hello"
text = "hello world"
match = re.search(pattern, text)
if match:
    print(f"找到匹配：{match.group()}")

# 常用元字符
# . 匹配任意字符（除换行）
# ^ 匹配字符串开头
# $ 匹配字符串结尾
# * 匹配前面的字符0次或多次
# + 匹配前面的字符1次或多次
# ? 匹配前面的字符0次或1次
# {n} 匹配前面的字符n次
# {n,} 匹配前面的字符至少n次
# {n,m} 匹配前面的字符n到m次
# [] 匹配括号内的任意一个字符
# | 匹配左边或右边的表达式
# () 分组

# 例子
pattern = r"^\d+$")  # 匹配纯数字
text = "12345"
if re.match(pattern, text):
    print("是纯数字")

# 分组
pattern = r"(\d{3})-(\d{4})-(\d{4})"  # 匹配电话号码
text = "138-1234-5678"
match = re.search(pattern, text)
if match:
    print(f"完整匹配：{match.group()}")
    print(f"区号：{match.group(1)}")
    print(f"中间4位：{match.group(2)}")
    print(f"后4位：{match.group(3)}")

# 常用方法
# re.search() - 查找第一个匹配
# re.match() - 从字符串开头匹配
# re.findall() - 查找所有匹配
# re.sub() - 替换匹配
# re.split() - 根据匹配分割

# 例子：提取所有数字
text = "价格：199元，数量：5个"
numbers = re.findall(r"\d+", text)
print(numbers)  # ['199', '5']

# 例子：替换
text = "hello 123 world 456"
result = re.sub(r"\d+", "***", text)
print(result)  # "hello *** world ***"
```
