## 第21章：无监督学习算法 🔍

### 21.1 无监督学习简介

无监督学习是指从未标记的训练数据中学习，算法需要自己发现数据中的模式。

> **核心任务**：
> - 聚类：将相似的数据点分组
> - 降维：减少数据的维度，同时保留重要信息
> - 关联规则学习：发现数据中项之间的关联

### 21.2 聚类算法

聚类是将相似的数据点分组的过程，使得同一组内的数据点相似度高，不同组之间的数据点相似度低。

#### 21.2.1 K-means 聚类

K-means 是最常用的聚类算法之一，它通过迭代优化来寻找K个聚类中心。

```python
from sklearn.cluster import KMeans
import numpy as np
import matplotlib.pyplot as plt

# 创建数据
np.random.seed(42)
X = np.random.randn(100, 2)

# 训练模型
kmeans = KMeans(n_clusters=3, random_state=42)
kmeans.fit(X)

# 预测
labels = kmeans.labels_
centers = kmeans.cluster_centers_

# 可视化
plt.scatter(X[:, 0], X[:, 1], c=labels, cmap='viridis')
plt.scatter(centers[:, 0], centers[:, 1], s=200, c='red', marker='X')
plt.title('K-means Clustering')
plt.xlabel('Feature 1')
plt.ylabel('Feature 2')
plt.show()

# 评估聚类效果
from sklearn.metrics import silhouette_score

score = silhouette_score(X, labels)
print(f"Silhouette Score: {score}")
```

#### 21.2.2 层次聚类

层次聚类通过构建层次结构来进行聚类，可以是自底向上（凝聚式）或自顶向下（分裂式）。

```python
from sklearn.cluster import AgglomerativeClustering
import numpy as np
import matplotlib.pyplot as plt
from scipy.cluster.hierarchy import dendrogram, linkage

# 创建数据
np.random.seed(42)
X = np.random.randn(100, 2)

# 凝聚式层次聚类
agg_clustering = AgglomerativeClustering(n_clusters=3, linkage='ward')
labels = agg_clustering.fit_predict(X)

# 可视化
plt.scatter(X[:, 0], X[:, 1], c=labels, cmap='viridis')
plt.title('Agglomerative Clustering')
plt.xlabel('Feature 1')
plt.ylabel('Feature 2')
plt.show()

# 绘制树状图
linked = linkage(X, 'ward')
plt.figure(figsize=(10, 7))
dendrogram(linked)
plt.title('Dendrogram')
plt.xlabel('Samples')
plt.ylabel('Distance')
plt.show()
```

#### 21.2.3 DBSCAN 聚类

DBSCAN（Density-Based Spatial Clustering of Applications with Noise）是一种基于密度的聚类算法，它可以发现任意形状的聚类，并且能够识别噪声点。

```python
from sklearn.cluster import DBSCAN
import numpy as np
import matplotlib.pyplot as plt

# 创建数据
np.random.seed(42)
X = np.random.randn(100, 2)

# 训练模型
dbscan = DBSCAN(eps=0.3, min_samples=5)
labels = dbscan.fit_predict(X)

# 可视化
plt.scatter(X[:, 0], X[:, 1], c=labels, cmap='viridis')
plt.title('DBSCAN Clustering')
plt.xlabel('Feature 1')
plt.ylabel('Feature 2')
plt.show()

# 评估聚类效果
from sklearn.metrics import silhouette_score

# 过滤掉噪声点（标签为-1）
if -1 in labels:
    mask = labels != -1
    score = silhouette_score(X[mask], labels[mask])
else:
    score = silhouette_score(X, labels)
print(f"Silhouette Score: {score}")
```

#### 21.2.4 OPTICS 聚类

OPTICS（Ordering Points To Identify the Clustering Structure）是DBSCAN的改进算法，它可以处理密度不均匀的数据集。

```python

import numpy as np
import matplotlib.pyplot as plt

# 可视化
plt.scatter(X[:, 0], X[:, 1], c=labels, cmap='viridis')
```

### 21.3 降维算法

降维是减少数据维度的过程，同时保留数据中的重要信息。

#### 21.3.1 主成分分析（PCA）

PCA是一种线性降维方法，它通过找到数据的主成分来减少维度。

```python
from sklearn.decomposition import PCA
import numpy as np
import matplotlib.pyplot as plt
from sklearn.datasets import load_iris

# 加载数据
data = load_iris()
X, y = data.data, data.target

# 应用PCA
pca = PCA(n_components=2)
X_pca = pca.fit_transform(X)

# 解释方差比例
print(f"Explained variance ratio: {pca.explained_variance_ratio_}")
print(f"Total explained variance: {sum(pca.explained_variance_ratio_)}")

# 可视化
plt.scatter(X_pca[:, 0], X_pca[:, 1], c=y, cmap='viridis')
plt.title('PCA of Iris Dataset')
plt.xlabel('PC1')
plt.ylabel('PC2')
plt.colorbar(ticks=[0, 1, 2], label='Species')
plt.show()
```

#### 21.3.2 t-SNE

t-SNE（t-Distributed Stochastic Neighbor Embedding）是一种非线性降维方法，特别适合可视化高维数据。

```python
from sklearn.manifold import TSNE
import numpy as np
import matplotlib.pyplot as plt
from sklearn.datasets import load_iris

# 加载数据
data = load_iris()
X, y = data.data, data.target

# 应用t-SNE
tsne = TSNE(n_components=2, perplexity=30, random_state=42)

# 可视化
plt.scatter(X_tsne[:, 0], X_tsne[:, 1], c=y, cmap='viridis')
plt.title('t-SNE of Iris Dataset')
plt.xlabel('t-SNE 1')
plt.ylabel('t-SNE 2')

plt.show()
```

#### 21.3.3 线性判别分析（LDA）

LDA是一种监督式降维方法，它通过最大化类间方差和最小化类内方差来降维。

```python
from sklearn.discriminant_analysis import LinearDiscriminantAnalysis
import numpy as np
import matplotlib.pyplot as plt
from sklearn.datasets import load_iris

# 加载数据
data = load_iris()
X, y = data.data, data.target

# 应用LDA
alda = LinearDiscriminantAnalysis(n_components=2)
X_lda = lda.fit_transform(X, y)

# 解释方差比例
print(f"Explained variance ratio: {lda.explained_variance_ratio_}")

# 可视化
plt.scatter(X_lda[:, 0], X_lda[:, 1], c=y, cmap='viridis')
plt.title('LDA of Iris Dataset')
plt.xlabel('LD1')
plt.ylabel('LD2')
plt.colorbar(ticks=[0, 1, 2], label='Species')
plt.show()
```

### 21.4 关联规则学习

关联规则学习是发现数据中项之间关联的过程，常用于购物篮分析。

#### 21.4.1 Apriori 算法

Apriori是一种经典的关联规则学习算法，它通过逐层搜索来发现频繁项集。

```python
from mlxtend.frequent_patterns import apriori, association_rules
import pandas as pd

# 创建示例数据
data = {
    'Transaction': [1, 2, 3, 4, 5],
    'Items': [
        ['牛奶', '面包', '鸡蛋'],
        ['牛奶', '面包', '尿布', '啤酒', '鸡蛋'],
        ['牛奶', '尿布', '啤酒', '可乐'],
        ['面包', '尿布', '啤酒', '橙汁'],
        ['面包', '尿布', '可乐']
    ]
}

# 转换为 one-hot 编码
from mlxtend.preprocessing import TransactionEncoder

te = TransactionEncoder()
X = te.fit_transform(data['Items'])
df = pd.DataFrame(X, columns=te.columns_)

# 应用 Apriori 算法
frequent_itemsets = apriori(df, min_support=0.4, use_colnames=True)
print("Frequent Itemsets:")
print(frequent_itemsets)

# 生成关联规则
rules = association_rules(frequent_itemsets, metric="confidence", min_threshold=0.7)
print("\nAssociation Rules:")
print(rules[['antecedents', 'consequents', 'support', 'confidence', 'lift']])
```

#### 21.4.2 FP-Growth 算法

FP-Growth是一种更高效的关联规则学习算法，它通过构建FP树来发现频繁项集。

```python
from mlxtend.frequent_patterns import fpgrowth
import pandas as pd
from mlxtend.preprocessing import TransactionEncoder

# 使用上面的数据
frequent_itemsets = fpgrowth(df, min_support=0.4, use_colnames=True)
print("Frequent Itemsets (FP-Growth):")
print(frequent_itemsets)

# 生成关联规则
rules = association_rules(frequent_itemsets, metric="confidence", min_threshold=0.7)
print("\nAssociation Rules:")
print(rules[['antecedents', 'consequents', 'support', 'confidence', 'lift']])
```

### 21.5 异常检测

异常检测是识别数据中异常或离群点的过程。

#### 21.5.1 孤立森林（Isolation Forest）

孤立森林是一种基于树的异常检测算法，它通过隔离异常点来识别异常。

```python
from sklearn.ensemble import IsolationForest
import numpy as np
import matplotlib.pyplot as plt

# 创建数据
np.random.seed(42)
normal_data = np.random.randn(100, 2)
anomaly_data = np.random.uniform(low=-4, high=4, size=(10, 2))
X = np.vstack([normal_data, anomaly_data])
y = np.zeros(len(X))
y[-10:] = 1  # 标记异常点

# 训练模型
iso_forest = IsolationForest(contamination=0.1, random_state=42)
pred = iso_forest.fit_predict(X)

# 预测结果：1表示正常，-1表示异常
anomalies = X[pred == -1]

# 可视化
plt.scatter(X[:, 0], X[:, 1], c='blue', label='Normal')
plt.scatter(anomalies[:, 0], anomalies[:, 1], c='red', label='Anomaly')
plt.title('Isolation Forest Anomaly Detection')
plt.xlabel('Feature 1')
plt.ylabel('Feature 2')
plt.legend()
plt.show()
```

#### 21.5.2 局部异常因子（Local Outlier Factor）

局部异常因子是一种基于密度的异常检测算法，它通过计算局部密度偏差来识别异常。

```python
from sklearn.neighbors import LocalOutlierFactor
import numpy as np
import matplotlib.pyplot as plt

# 使用上面的数据
lof = LocalOutlierFactor(n_neighbors=20, contamination=0.1)
pred = lof.fit_predict(X)

# 预测结果：1表示正常，-1表示异常

# 可视化

plt.scatter(anomalies[:, 0], anomalies[:, 1], c='red', label='Anomaly')
plt.title('Local Outlier Factor Anomaly Detection')
plt.xlabel('Feature 1')
plt.ylabel('Feature 2')
plt.legend()
plt.show()
```

### 21.6 无监督学习的评估

无监督学习的评估比监督学习更困难，因为没有真实标签。

#### 21.6.1 聚类评估指标

- **轮廓系数（Silhouette Score）**：衡量聚类的质量，值越接近1越好
- **Calinski-Harabasz 指数**：衡量聚类的紧密度和分离度，值越大越好
- **Davies-Bouldin 指数**：衡量聚类的平均相似度，值越小越好

```python
from sklearn.metrics import silhouette_score, calinski_harabasz_score, davies_bouldin_score
from sklearn.cluster import KMeans
import numpy as np

# 创建数据
np.random.seed(42)
X = np.random.randn(100, 2)

# 训练模型
kmeans = KMeans(n_clusters=3, random_state=42)
labels = kmeans.fit_predict(X)

# 评估
print(f"Silhouette Score: {silhouette_score(X, labels)}")
print(f"Calinski-Harabasz Score: {calinski_harabasz_score(X, labels)}")
print(f"Davies-Bouldin Score: {davies_bouldin_score(X, labels)}")
```

#### 21.6.2 降维评估指标

- **解释方差比例**：衡量降维后保留的信息量，值越接近1越好
- **重构误差**：衡量降维后重构原始数据的误差，值越小越好

```python
from sklearn.decomposition import PCA
from sklearn.datasets import load_iris
import numpy as np

# 加载数据
data = load_iris()
X = data.data

# 应用PCA
pca = PCA(n_components=2)
X_pca = pca.fit_transform(X)

# 重构数据
X_reconstructed = pca.inverse_transform(X_pca)

# 计算重构误差
reconstruction_error = np.mean((X - X_reconstructed) ** 2)
print(f"Reconstruction Error: {reconstruction_error}")
print(f"Explained Variance Ratio: {sum(pca.explained_variance_ratio_)}")
```

### 21.7 无监督学习的应用

1. **客户分群**：根据客户的购买行为、 demographics 等信息将客户分成不同的群体
2. **异常检测**：识别信用卡欺诈、网络入侵、设备故障等异常情况
3. **数据可视化**：将高维数据降维后可视化，发现数据中的模式
4. **特征学习**：从无标记数据中学习有用的特征表示
5. **推荐系统**：基于用户的行为模式推荐产品或内容
6. **文本聚类**：将文档或文本分成不同的主题或类别
7. **图像分割**：将图像中的像素分成不同的区域

### 21.8 无监督学习算法选择指南

| 任务类型 | 推荐算法 | 特点 |
|---------|---------|------|
| 聚类 | K-means | 简单高效，适用于球形聚类 |
| 聚类 | DBSCAN | 能发现任意形状的聚类，能识别噪声 |
| 聚类 | 层次聚类 | 提供聚类的层次结构，可视化效果好 |
| 降维 | PCA | 线性降维，计算效率高 |
| 降维 | t-SNE | 非线性降维，可视化效果好 |
| 降维 | UMAP | 结合了PCA的效率和t-SNE的效果 |
| 关联规则 | Apriori | 经典算法，易于理解 |
| 关联规则 | FP-Growth | 更高效，适用于大规模数据 |
| 异常检测 | 孤立森林 | 适用于高维数据，计算效率高 |
| 异常检测 | 局部异常因子 | 基于密度，适用于局部异常 |
