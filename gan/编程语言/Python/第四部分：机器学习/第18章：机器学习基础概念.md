## 第18章：机器学习基础概念 🤖

### 18.1 什么是机器学习？

机器学习是人工智能的一个分支，它使计算机能够从数据中学习，而不需要被显式编程。

> **核心思想**：通过学习数据中的模式，使计算机能够对新数据做出预测或决策。

### 18.2 机器学习的类型

#### 18.2.1 监督学习

监督学习是指从标记的训练数据中学习，每个训练样本都有输入和对应的输出。

- **分类**：预测离散的类别标签（如垃圾邮件检测、图像分类）
- **回归**：预测连续的数值（如房价预测、股票价格预测）
- **常见算法**：线性回归、逻辑回归、决策树、随机森林、支持向量机、K最近邻

#### 18.2.2 无监督学习

无监督学习是指从未标记的训练数据中学习，算法需要自己发现数据中的模式。

- **聚类**：将相似的数据点分组（如客户分群、异常检测）
- **降维**：减少数据的维度，同时保留重要信息（如PCA）
- **关联规则学习**：发现数据中项之间的关联（如购物篮分析）
- **常见算法**：K-means聚类、层次聚类、DBSCAN、PCA、t-SNE

#### 18.2.3 半监督学习

半监督学习结合了监督学习和无监督学习，使用少量标记数据和大量未标记数据。

#### 18.2.4 强化学习

强化学习是指智能体通过与环境交互，学习如何采取行动以最大化累积奖励。

- **应用**：游戏AI、机器人控制、推荐系统
- **常见算法**：Q-learning、深度Q网络（DQN）、策略梯度

### 18.3 机器学习的工作流程

1. **数据收集**：获取原始数据
2. **数据预处理**：清洗、转换、特征工程
3. **数据分割**：训练集、验证集、测试集
4. **模型选择**：选择合适的算法
5. **模型训练**：使用训练数据拟合模型
6. **模型评估**：使用验证集评估模型性能
7. **模型调优**：超参数调优
8. **模型部署**：将模型应用到实际场景

### 18.4 数据预处理

#### 18.4.1 数据清洗

- **处理缺失值**：删除、填充（均值、中位数、众数）
- **处理异常值**：删除、替换
- **处理重复值**：删除重复数据

#### 18.4.2 特征工程

- **特征选择**：选择对预测有帮助的特征
- **特征转换**：标准化、归一化、编码（独热编码、标签编码）
- **特征提取**：从原始数据中提取新特征

### 18.5 模型评估指标

#### 18.5.1 分类评估指标

- **准确率（Accuracy）**：正确预测的样本数占总样本数的比例
- **精确率（Precision）**：预测为正类的样本中实际为正类的比例
- **召回率（Recall）**：实际为正类的样本中被正确预测的比例
- **F1分数**：精确率和召回率的调和平均
- **混淆矩阵**：显示分类结果的矩阵
- **ROC曲线**：Receiver Operating Characteristic曲线，评估分类模型的性能
- **AUC**：ROC曲线下的面积，值越大模型性能越好

#### 18.5.2 回归评估指标

- **均方误差（MSE）**：预测值与真实值之差的平方的平均值
- **均方根误差（RMSE）**：MSE的平方根
- **平均绝对误差（MAE）**：预测值与真实值之差的绝对值的平均值
- **R²分数**：模型解释的方差比例，越接近1越好

### 18.6 过拟合与欠拟合

- **过拟合**：模型在训练数据上表现很好，但在新数据上表现很差
  - **原因**：模型过于复杂，学习了训练数据中的噪声
  - **解决方法**：正则化、减少特征、增加数据、交叉验证

- **欠拟合**：模型在训练数据和新数据上表现都很差
  - **原因**：模型过于简单，无法捕捉数据中的模式
  - **解决方法**：增加模型复杂度、增加特征、减少正则化

### 18.7 交叉验证

交叉验证是一种评估模型性能的方法，通过将数据分为多个子集，轮流使用不同的子集作为验证集。

#### 18.7.1 K折交叉验证

```python
from sklearn.model_selection import cross_val_score

# K折交叉验证
scores = cross_val_score(model, X, y, cv=5)
print(f"Cross-validation scores: {scores}")
print(f"Mean score: {scores.mean()}")
```

#### 18.7.2 留一交叉验证

留一交叉验证是K折交叉验证的特殊情况，其中K等于样本数。

#### 18.7.3 留P交叉验证

留P交叉验证是指每次留下P个样本作为验证集。

### 18.8 超参数调优

超参数是在模型训练前设置的参数，需要通过调优来找到最佳值。

#### 18.8.1 网格搜索

```python
from sklearn.model_selection import GridSearchCV

# 定义参数网格
param_grid = {
    'n_estimators': [100, 200, 300],
    'max_depth': [3, 5, 7]
}

# 网格搜索
grid_search = GridSearchCV(estimator=model, param_grid=param_grid, cv=5)
grid_search.fit(X_train, y_train)

# 最佳参数
print(f"Best parameters: {grid_search.best_params_}")
print(f"Best score: {grid_search.best_score_}")
```

#### 18.8.2 随机搜索

随机搜索是指在参数空间中随机采样进行搜索，适用于参数较多的情况。

```python
from sklearn.model_selection import RandomizedSearchCV

# 随机搜索
random_search = RandomizedSearchCV(estimator=model, param_distributions=param_distributions, n_iter=10, cv=5)
random_search.fit(X_train, y_train)
```

### 18.9 特征选择

特征选择是选择对预测有帮助的特征，减少维度，提高模型性能。

#### 18.9.1 过滤法

- **方差阈值**：删除方差低于阈值的特征
- **相关系数**：选择与目标变量相关性高的特征
- **卡方检验**：选择与目标变量相关的特征

#### 18.9.2 包装法

- **递归特征消除（RFE）**：递归删除最不重要的特征
- **特征重要性**：基于模型的特征重要性选择特征

#### 18.9.3 嵌入法

- **L1正则化**：通过L1正则化自动选择特征
- **决策树**：通过决策树的特征重要性选择特征

### 18.10 模型集成

模型集成是将多个模型的预测结果组合起来，以提高预测性能。

#### 18.10.1 投票法

```python
from sklearn.ensemble import VotingClassifier

# 投票分类器
voting_clf = VotingClassifier(
    estimators=[
        ('lr', LogisticRegression()),
        ('dt', DecisionTreeClassifier()),
        ('rf', RandomForestClassifier())
    ],
    voting='hard'  # 'hard' 或 'soft'
)

voting_clf.fit(X_train, y_train)
```

#### 18.10.2  Bagging

Bagging（Bootstrap Aggregating）是指从原始数据中随机采样，训练多个模型，然后取平均值。

```python
from sklearn.ensemble import BaggingClassifier

# Bagging分类器
bagging_clf = BaggingClassifier(
    base_estimator=DecisionTreeClassifier(),
    n_estimators=100,
    random_state=42
)

bagging_clf.fit(X_train, y_train)
```

#### 18.10.3 随机森林

随机森林是Bagging的一种变体，它在构建决策树时随机选择特征。

```python
from sklearn.ensemble import RandomForestClassifier

# 随机森林分类器
rf_clf = RandomForestClassifier(
    n_estimators=100,
    max_depth=5,
    random_state=42
)

rf_clf.fit(X_train, y_train)
```

#### 18.10.4 Boosting

Boosting是指迭代训练多个弱学习器，每个学习器都关注前一个学习器错误分类的样本。

- **AdaBoost**：自适应 boosting
- **Gradient Boosting**：梯度提升
- **XGBoost**：极端梯度提升
- **LightGBM**：轻量级梯度提升机
- **CatBoost**：类别特征的梯度提升
