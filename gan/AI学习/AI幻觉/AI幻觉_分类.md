# AI 幻觉：分类

> 📌 来源：学术研究论文

---

## 按与事实的关系分类

```
┌─────────────────────────────────────────────┐
│              AI 幻觉分类                     │
├───────────────────┬─────────────────────────┤
│  内在幻觉          │  外在幻觉                │
│ (Intrinsic)       │  (Extrinsic)             │
│                   │                          │
│ 输出与来源材料     │ 输出内容无法从来源验证   │
│ 直接矛盾           │ 也不直接矛盾             │
│                   │                          │
│ 例：总结文章时     │ 例：模型自行添加了       │
│ 改变了原文的意思   │ 原文中没有的细节         │
└───────────────────┴─────────────────────────┘
```

---

## 按问题域分类

| 类型 | 特征 | 例子 |
|------|------|------|
| **闭域幻觉** | 输出与问题前提矛盾 | 你说"苹果是红色的"，AI 回复"是的，苹果是绿色的" |
| **开放域幻觉** | 回答内容无法验证 | 编造一个不存在的历史人物的生平 |
| **事实幻觉** | 错误的具体事实 | 张冠李戴，把 A 人的成就归到 B 人名下 |
| **引用幻觉** | 编造不存在的参考文献 | 给出一篇"真实期刊"中的"真实论文"但完全是捏造的 |
