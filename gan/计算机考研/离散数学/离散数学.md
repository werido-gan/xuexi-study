### 1.1 命题的基本概念

#### 🔑 什么是命题？

**命题** = 能判断真假的陈述句（非疑问句、感叹句、祈使句）

| 例子 | 是否命题 | 原因 |
|------|----------|------|
| 北京是中国的首都 | ✅ 是 | 陈述句，真值为T |
| 1+1=3 | ✅ 是 | 陈述句，真值为F |
| 请关灯！ | ❌ 否 | 祈使句 |
| x > 5 | ❌ 否 | x未定，无法判断真假 |
| 这句话是假的 | ❌ 否 | 悖论，无法赋真值 |

> 💡 **小白记忆法**：命题就像"判断题"——要么对要么错，必须有一个确定答案。

#### 五个联结词详解

**1. 否定（¬p）**

| p | ¬p |
|---|-----|
| T | F |
| F | T |

直觉理解：¬p = "不是p"

---

**2. 合取（p ∧ q）**

| p | q | p ∧ q |
|---|---|--------|
| T | T | **T** |
| T | F | F |
| F | T | F |
| F | F | F |

直觉理解：两个都真才为真（"且"，像"AND"门）

---

**3. 析取（p ∨ q）**

| p | q | p ∨ q |
|---|---|--------|
| T | T | T |
| T | F | T |
| F | T | T |
| F | F | **F** |

直觉理解：至少一个为真则为真（"或"，像"OR"门）

---

**4. 蕴含（p → q）⚠️ 最易出错！**

| p | q | p → q |
|---|---|--------|
| T | T | T |
| T | F | **F** |
| F | T | **T** |
| F | F | **T** |

> 🔥 **关键口诀**：蕴含只有"真推假"才为假！
> 
> 生活类比：老师说"如果你考100分，我就请你吃饭"（p→q）
> - p=T（考了100分），q=T（请吃饭了）→ T ✓
> - p=T（考了100分），q=F（没请吃饭）→ F ✗（老师失言）
> - p=F（没考100分），q=T（请吃饭了）→ T（额外奖励，没说不行）
> - p=F（没考100分），q=F（没请吃饭）→ T（老师没说错）

**蕴含的重要变形：**
- 逆命题：q → p
- 否命题：¬p → ¬q
- 逆否命题：¬q → ¬p（与原命题等价！）

---

**5. 双条件（p ↔ q）**

| p | q | p ↔ q |
|---|---|--------|
| T | T | **T** |
| T | F | F |
| F | T | F |
| F | F | **T** |

直觉理解：同真同假则为真（"当且仅当"）

---

### 1.2 命题公式与分类

**命题公式** 是由命题变元和联结词构成的表达式。

#### 三种类型

```
                      真值表
                     /       \
            全为T              有T有F
         （永真式）                |
           重言式         至少一个T
                         （可满足式）
                              |
                          全为F
                         （矛盾式）
                          永假式
```

**判断方法**：列真值表（n个变元需 2ⁿ 行）

#### 🔑 运算优先级（从高到低）
1. `¬` （否定，最高）
2. `∧` （合取）
3. `∨` （析取）
4. `→` （蕴含）
5. `↔` （双条件，最低）

> 💡 记忆法：**¬∧∨→↔**，就像算术里先乘除后加减

### 1.3 等值演算

#### 重要等值式（必须熟背！）

**基本律：**

| 名称 | 公式 |
|------|------|
| 双重否定律 | ¬¬p ≡ p |
| 幂等律 | p ∧ p ≡ p，p ∨ p ≡ p |
| 交换律 | p ∧ q ≡ q ∧ p，p ∨ q ≡ q ∨ p |
| 结合律 | (p∧q)∧r ≡ p∧(q∧r) |
| 分配律 | p∧(q∨r) ≡ (p∧q)∨(p∧r) |
| 德摩根律 | ¬(p∧q) ≡ ¬p∨¬q，¬(p∨q) ≡ ¬p∧¬q |
| 吸收律 | p∧(p∨q) ≡ p，p∨(p∧q) ≡ p |
| 零律 | p∧F ≡ F，p∨T ≡ T |
| 同一律 | p∧T ≡ p，p∨F ≡ p |
| 排中律 | p∨¬p ≡ T |
| 矛盾律 | p∧¬p ≡ F |

**蕴含相关：**

| 公式 | 说明 |
|------|------|
| p→q ≡ ¬p∨q | 最重要！蕴含转化为析取 |
| p→q ≡ ¬q→¬p | 逆否命题等价 |
| p↔q ≡ (p→q)∧(q→p) | 双条件分解 |
| p↔q ≡ (p∧q)∨(¬p∧¬q) | 另一种分解 |

> 🔥 **德摩根律生活理解**：
> - "不是（既下雨又刮风）" = "不下雨 或 不刮风"
> - "不是（下雨或刮风）" = "不下雨 且 不刮风"

### 1.4 范式

**为什么需要范式？** 给公式一个"标准形式"，便于比较和分析。

#### 析取范式（DNF = Disjunctive Normal Form）

**形式**：若干个合取式的析取  
`(子句1) ∨ (子句2) ∨ ... ∨ (子句n)`

其中每个子句是若干文字（变元或其否定）的合取。

**例**：`(p ∧ ¬q) ∨ (¬p ∧ q) ∨ (p ∧ q)` 是DNF

#### 合取范式（CNF = Conjunctive Normal Form）

**形式**：若干个析取式的合取  
`(子句1) ∧ (子句2) ∧ ... ∧ (子句n)`

其中每个子句是若干文字的析取。

**例**：`(p ∨ ¬q) ∧ (¬p ∨ q)` 是CNF

#### 求范式的步骤

```
原公式
  ↓ 步骤1：消去 → 和 ↔
     p→q ≡ ¬p∨q
     p↔q ≡ (¬p∨q)∧(p∨¬q)
  ↓ 步骤2：德摩根律将¬移到变元内
     ¬(A∧B) ≡ ¬A∨¬B
     ¬(A∨B) ≡ ¬A∧¬B
     ¬¬A ≡ A
  ↓ 步骤3：用分配律展开
     DNF：∧对∨分配
     CNF：∨对∧分配
```

#### 主范式（标准范式）

**极小项**（针对DNF）：每个变元都出现且恰好出现一次的合取式  
**极大项**（针对CNF）：每个变元都出现且恰好出现一次的析取式

对于n个变元，有 2ⁿ 个极小项，编号为 m₀, m₁, ..., m_{2ⁿ-1}

> 💡 编号技巧：变元取T=1，取F=0，按顺序组成二进制数，即为极小项编号

---

### 1.5 联结词全功能集

**全功能集** = 用这个集合的联结词可以表达所有逻辑功能

- `{¬, ∧, ∨}` 是全功能集
- `{¬, ∧}` 是全功能集（因为 p∨q ≡ ¬(¬p∧¬q)）
- `{¬, ∨}` 是全功能集
- `{↓}` 是全功能集（NOR，与非）
- `{↑}` 是全功能集（NAND，或非）

其中 `↑`（与非，Sheffer stroke）：p↑q ≡ ¬(p∧q)

### 1.6 组合电路

命题逻辑与数字电路一一对应：

| 逻辑 | 电路 |
|------|------|
| ¬p | NOT门 |
| p∧q | AND门 |
| p∨q | OR门 |
| p↑q | NAND门 |
| p↓q | NOR门 |

**加法器示例**：
- 半加器：sum = p⊕q，carry = p∧q
- 其中 p⊕q（异或）= (p∨q)∧¬(p∧q)

### 1.7 推理理论

#### 推理的形式结构

**有效推理**：若 (H₁ ∧ H₂ ∧ ... ∧ Hₙ) → C 是永真式，则称由前提 H₁,...,Hₙ 推出结论 C 是有效的。

记作：H₁, H₂, ..., Hₙ ⊢ C

#### 基本推理规则

| 规则名 | 形式 | 说明 |
|--------|------|------|
| 假言推理（MP） | p, p→q ⊢ q | 肯定前件 |
| 假言三段论（HS） | p→q, q→r ⊢ p→r | 推理链条 |
| 化简（Simp） | p∧q ⊢ p | 取出合取项 |
| 附加（Add） | p ⊢ p∨q | 加入析取项 |
| 合取引入（Conj） | p, q ⊢ p∧q | 合并 |
| 拒取式（MT） | ¬q, p→q ⊢ ¬p | 否定后件 |
| 析取三段论（DS） | p∨q, ¬p ⊢ q | 排除 |

#### 两种证明方法

**直接证明法**：从前提直接推出结论

**间接证明法（反证法）**：假设结论的否定，与前提一起推出矛盾（假即可）

> 💡 **选择技巧**：
> - 结论为蕴含式 p→q 时，可附加 p 作为新前提，证明 q
> - 看到矛盾好怀疑时，用归谬法（反证法）

#### 例题精讲

**例**：证明 p→q, ¬q∨r, ¬r ⊢ ¬p

| 步骤 | 公式 | 理由 |
|------|------|------|
| 1 | ¬q∨r | 前提2 |
| 2 | ¬r | 前提3 |
| 3 | ¬q | DS（步骤1,2：消去r，得¬q） |
| 4 | p→q | 前提1 |
| 5 | ¬p | MT（步骤3,4：拒取式） |

### 2.1 一阶逻辑基本概念

#### 为什么命题逻辑不够用？

命题逻辑只能处理完整命题，无法表达：
- "**所有**人都会死"（全称）
- "**存在**一个质数大于100"（存在）
- "苏格拉底是人" → "苏格拉底会死"（内部结构）

一阶逻辑（谓词逻辑）引入了以下工具：

#### 个体与论域

- **个体**：研究对象（论域中的元素），通常用 a, b, c 表示常量，x, y, z 表示变量
- **论域**（个体域）：所有研究对象的集合

#### 谓词

**谓词**描述个体的性质或个体间的关系：
- 一元谓词：`P(x)` 表示"x是..."，如 M(x) 表示"x是人"
- 二元谓词：`L(x,y)` 表示"x和y的关系"，如 L(x,y) 表示"x爱y"
- n元谓词：有n个参数

**例子：**
- 苏格拉底是人：`M(s)`（s为苏格拉底）
- 苏格拉底爱柏拉图：`L(s, p)`
- x大于y：`G(x, y)` 或写为 x > y

#### 量词

**全称量词 ∀**（Universal Quantifier）

`∀xP(x)` 读作"对所有x，P(x)成立"  
等价于：论域中每个个体都满足P

**存在量词 ∃**（Existential Quantifier）

`∃xP(x)` 读作"存在一个x，使P(x)成立"  
等价于：论域中至少有一个个体满足P

#### 自由变元与约束变元

```
∀x (P(x) → Q(x,y))
       |          |
   约束变元x    自由变元y

x被∀x约束，y没有被量词约束（自由出现）
```

**规则：**
- 约束变元可以改名（α-变换）：`∀xP(x) ≡ ∀yP(y)`
- 自由变元影响公式真值

### 2.2 合式公式（WFF）

#### 构成规则（递归定义）

1. 原子公式（谓词加个体项）是WFF
2. 若A是WFF，则¬A是WFF
3. 若A,B是WFF，则A∧B, A∨B, A→B, A↔B是WFF
4. 若A是WFF，x是变元，则∀xA, ∃xA是WFF

#### 量词辖域

量词的辖域 = 量词紧跟的那个WFF（括号内的部分）

```
∀x P(x) → Q(x)
    ↑辖域↑
（辖域只包括P(x)，后面的Q(x)中x是自由变元！）

∀x (P(x) → Q(x))
        辖域包括整个括号内↑
```

### 2.3 一阶逻辑等值式

#### 量词否定律（重要！）

```
¬∀xA(x) ≡ ∃x¬A(x)    "不是所有x都A" = "存在x不A"
¬∃xA(x) ≡ ∀x¬A(x)    "不存在x使A" = "所有x都不A"
```

> 💡 **生活理解**：
> - "不是所有学生都聪明" = "存在不聪明的学生"（正确！）
> - "不存在会飞的猪" = "所有猪都不会飞"（正确！）

#### 量词与联结词

```
∀x(A(x)∧B(x)) ≡ ∀xA(x) ∧ ∀xB(x)    ✅（全称+合取可分配）
∃x(A(x)∨B(x)) ≡ ∃xA(x) ∨ ∃xB(x)    ✅（存在+析取可分配）

∀x(A(x)∨B(x)) ≢ ∀xA(x) ∨ ∀xB(x)   ❌（全称+析取不能分配！）
∃x(A(x)∧B(x)) ≢ ∃xA(x) ∧ ∃xB(x)   ❌（存在+合取不能分配！）
```

> ⚠️ **易错点**：全称和析取、存在和合取**不能**随便分配！

#### 量词辖域扩张（x不在B中自由出现）

```
∀xA(x) ∧ B ≡ ∀x(A(x) ∧ B)
∀xA(x) ∨ B ≡ ∀x(A(x) ∨ B)
∃xA(x) ∧ B ≡ ∃x(A(x) ∧ B)
∃xA(x) ∨ B ≡ ∃x(A(x) ∨ B)
```

#### 多个量词的交换

```
∀x∀yA(x,y) ≡ ∀y∀xA(x,y)    ✅ 相同量词可交换
∃x∃yA(x,y) ≡ ∃y∃xA(x,y)    ✅ 相同量词可交换

∀x∃yA(x,y) ≢ ∃y∀xA(x,y)    ❌ 不同量词顺序不能交换！
```

> 💡 **反例**：
> - ∀x∃y(x<y)：对每个数x，存在比它大的y → 在整数上为真
> - ∃y∀x(x<y)：存在一个y比所有x都大 → 在整数上为假

### 2.4 前束范式

**前束范式**：所有量词都在公式最前面（量词前缀 + 母式）

形式：`Q₁x₁ Q₂x₂ ... Qₙxₙ M(x₁,...,xₙ)`

其中 Qᵢ 是 ∀ 或 ∃，M 是不含量词的公式。

#### 化为前束范式的步骤

1. 消去 → 和 ↔
2. 德摩根律将 ¬ 移入（同时改变量词：¬∀→∃¬，¬∃→∀¬）
3. 换名避免冲突
4. 将量词全部移到最前

**例题**：将 `¬∀xP(x) → ∃yQ(y)` 化为前束范式

```
¬∀xP(x) → ∃yQ(y)
≡ ¬¬∀xP(x) ∨ ∃yQ(y)          （消去→：A→B ≡ ¬A∨B）
≡ ∀xP(x) ∨ ∃yQ(y)             （双重否定）
≡ ∀x(P(x) ∨ ∃yQ(y))           （辖域扩张）
≡ ∀x∃y(P(x) ∨ Q(y))           （辖域扩张）
```

---

### 一阶逻辑推理

一阶逻辑推理在命题逻辑推理规则基础上，增加量词的消去和引入规则：

| 规则 | 名称 | 说明 |
|------|------|------|
| ∀x A(x) ⊢ A(t) | 全称实例化（UI） | t是任意项 |
| A(t) ⊢ ∀xA(x) | 全称推广（UG） | 需t满足条件 |
| ∃x A(x) ⊢ A(c) | 存在实例化（EI） | c是特定常量 |
| A(c) ⊢ ∃xA(x) | 存在推广（EG） | c是任意常量 |

1. 将自然语言符号化（最难，最重要！）

3. 求前束范式

- 全称量词只能和∧分配，存在量词只能和∨分配

**自然语言→一阶逻辑的常用模式：**

| 中文表达 | 一阶逻辑 |
|----------|----------|

### 3.1 集合基本概念

1. **枚举法**：`A = {1, 2, 3, 4, 5}`

若 |A| = n，则 |P(A)| = 2ⁿ

**例**：A = {a, b}  
P(A) = {∅, {a}, {b}, {a,b}}，共 2² = 4 个元素

> ⚠️ **注意**：∅ ∈ P(A)（空集是集合的元素，而不是子集）

### 3.3 容斥原理（元素计数）

**两个集合**：
|A∪B| = |A| + |B| - |A∩B|

**三个集合**：
|A∪B∪C| = |A|+|B|+|C| - |A∩B| - |B∩C| - |A∩C| + |A∩B∩C|

**通用公式（容斥原理）**：

|A₁∪A₂∪...∪Aₙ| = Σ|Aᵢ| - Σ|Aᵢ∩Aⱼ| + Σ|Aᵢ∩Aⱼ∩Aₖ| - ... + (-1)ⁿ⁺¹|A₁∩...∩Aₙ|

**例题**：  
100人中，60人喜欢A，50人喜欢B，30人同时喜欢A和B。  
问：喜欢A或B的有多少人？

解：|A∪B| = |A| + |B| - |A∩B| = 60 + 50 - 30 = **80人**

### 4.1 笛卡儿积与二元关系

#### 笛卡儿积（Cartesian Product）

A×B = {(a,b) | a∈A 且 b∈B}

**例**：A={1,2}, B={a,b}  
A×B = {(1,a),(1,b),(2,a),(2,b)}

**性质**：|A×B| = |A| × |B|

> ⚠️ 注意：A×B ≠ B×A（有序对顺序不同）

#### 二元关系

**关系** R 是 A×B 的子集：R ⊆ A×B

若 A = B，则称 R 是 A 上的关系。

**xRy** 表示 (x,y) ∈ R

#### 关系的表示方法

**1. 集合表示法**：R = {(a,b), (b,c), (a,c)}

**2. 关系矩阵（布尔矩阵）**：
```
M_R[i][j] = 1  若 (aᵢ,aⱼ) ∈ R
M_R[i][j] = 0  若 (aᵢ,aⱼ) ∉ R
```

**3. 关系图（有向图）**：
```
有向图：用箭头表示关系
a ──→ b 表示 (a,b) ∈ R
```

---

### 4.2 关系的运算

#### 关系的复合

S ∘ R = {(a,c) | 存在b，使 (a,b)∈R 且 (b,c)∈S}

> 💡 注意顺序！S∘R 先用R再用S（从右到左）

**矩阵表示**：M_{S∘R} = M_R ⊙ M_S（布尔矩阵乘法）

布尔矩阵乘法：c_{ij} = ∨ₖ(a_{ik} ∧ b_{kj})

#### 关系的幂

R¹ = R  
R^n = R^{n-1} ∘ R（n≥2）

---

### 4.3 关系的四大性质

| 性质 | 定义 | 矩阵判断 | 图判断 |
|------|------|----------|--------|
| 自反 | ∀x∈A: (x,x)∈R | 主对角线全1 | 每个结点有自环 |
| 对称 | (x,y)∈R → (y,x)∈R | M=M^T（对称矩阵）| 有边必有反向边 |
| 反对称 | (x,y)∈R且(y,x)∈R → x=y | i≠j时不能同时有1 | 无双向边（自环可有）|
| 传递 | (x,y)∈R,(y,z)∈R → (x,z)∈R | M²的1都在M中 | 有间接路径必有直接边 |

#### 关系性质判断表格

| 关系 | 自反 | 对称 | 反对称 | 传递 |
|------|------|------|--------|------|
| 整除关系（ℤ+上）| ✅ | ❌ | ✅ | ✅ |
| 小于关系（<）| ❌ | ❌ | ✅ | ✅ |
| 小于等于（≤）| ✅ | ❌ | ✅ | ✅ |
| 相等（=）| ✅ | ✅ | ✅ | ✅ |
| 同学关系 | ✅ | ✅ | ❌ | ✅ |

> 💡 **技巧**：对称和反对称不互斥！一个关系可以同时是对称和反对称的（如等于关系）

---

### 4.4 关系的闭包

**闭包** = 包含原关系的、满足某性质的最小关系

| 类型 | 构造方法 | 矩阵方法 |
|------|----------|----------|
| 自反闭包 r(R) | R ∪ Iₐ（加上单位关系）| M_R ∨ I（加1到主对角线） |
| 对称闭包 s(R) | R ∪ R⁻¹（加上逆关系）| M_R ∨ M_R^T |
| 传递闭包 t(R) | R∪R²∪R³∪...（Warshall算法）| 见下 |

#### Warshall算法求传递闭包

```python
# 输入：n×n布尔矩阵M
# 输出：传递闭包M+

M_plus = M  # 初始化
for k in range(n):
    for i in range(n):
        for j in range(n):
            M_plus[i][j] = M_plus[i][j] OR (M_plus[i][k] AND M_plus[k][j])
```

时间复杂度：O(n³)

### 4.5 等价关系与偏序关系

#### 等价关系

满足**自反+对称+传递**的关系称为等价关系，记为 ~

**等价类**：`[a] = {x ∈ A | x ~ a}`（所有与a等价的元素）

**商集**：`A/~ = {[a] | a ∈ A}`（所有等价类的集合）

**关键定理**：等价关系把集合分成互不相交的等价类（划分）

> 💡 **例子**：整数的模3等价关系
> - [0] = {...,-3, 0, 3, 6, ...}
> - [1] = {...,-2, 1, 4, 7, ...}
> - [2] = {...,-1, 2, 5, 8, ...}
> - ℤ/~ = {[0], [1], [2]}，三个等价类

#### 偏序关系

满足**自反+反对称+传递**的关系称为偏序关系，记为 ≤（或 ⪯）

**偏序集**：(A, ≤)，集合加偏序关系

**哈斯图（Hasse Diagram）**：偏序集的图示方法
- 去掉自反边（自环）
- 去掉传递得到的边
- 从下到上表示"小于"方向

```
{1,2,3,4,6,12}上的整除关系哈斯图：

     12
    /  \
   4    6
   |   /|
   2  2 3
    \ |
     1
```

**特殊元素：**
- **最大元**：比所有元素都大（唯一，如果存在）
- **最小元**：比所有元素都小（唯一，如果存在）
- **极大元**：没有比它更大的元素（可能多个）
- **极小元**：没有比它更小的元素（可能多个）
- **上界/下界**：子集的上界/下界
- **最小上界（上确界/lub）**：上界中最小的
- **最大下界（下确界/glb）**：下界中最大的

### 4.6 函数

#### 函数的定义

函数 f: A → B 是 A×B 的一个关系，满足：
- 对每个 a∈A，恰好存在唯一的 b∈B，使 (a,b)∈f

即：每个x有且只有一个对应的y

#### 函数的分类

| 类型 | 定义 | 判断 |
|------|------|------|
| 单射（一对一）| f(x)=f(y) → x=y | 不同输入有不同输出 |
| 满射（映上）| 对所有b∈B，∃a∈A使f(a)=b | 每个输出都被覆盖 |
| 双射（一一对应）| 既单射又满射 | 完美配对 |

```
单射：A=3个元素，B=5个元素，3条线不交叉
满射：A=5个元素，B=3个元素，3个都被射到
双射：A=3个元素，B=3个元素，完美一一对应
```

#### 复合函数

(g∘f)(x) = g(f(x))

> ⚠️ 顺序：先f后g（从右到左读）

#### 反函数

当 f: A→B 是双射时，f⁻¹: B→A 存在

f⁻¹(b) = a  当且仅当  f(a) = b

**关键性质：**
- f⁻¹∘f = I_A（A上的恒等函数）
- f∘f⁻¹ = I_B

### 5.1 图的基本定义

#### 无向图

G = (V, E)，其中：
- V：顶点集（非空）
- E：边集，每条边是两个顶点的无序对 {u,v}

#### 有向图

D = (V, A)，其中：
- V：顶点集
- A：弧集（有向边集），每条弧是有序对 (u,v)（从u到v）

#### 特殊图

| 名称 | 定义 |
|------|------|

### 5.2 度数与握手定理

**度数 d(v)**：顶点v关联的边的数目（有向图中分为入度 d⁻(v) 和出度 d⁺(v)）

#### 握手定理（最重要！）

**无向图**：Σ d(v) = 2|E|（所有顶点度数之和 = 边数的两倍）

**推论**：度数为奇数的顶点必有偶数个

**有向图**：Σ d⁺(v) = Σ d⁻(v) = |A|

### 5.3 通路与连通性

#### 通路的类型

| 名称 | 条件 |
|------|------|
| 通路（Walk） | 边和顶点可重复 |
| 迹（Trail） | 边不重复，顶点可重复 |
| 路径（Path） | 边和顶点都不重复 |
| 回路（Circuit/Closed Walk） | 始末顶点相同 |
| 简单回路（Simple Circuit）| 边不重复的回路 |
| 圈/基本回路（Cycle） | 顶点不重复的回路 |

#### 连通性

**无向图**：
- 连通图：任意两顶点间有通路
- 连通分量：极大连通子图

**有向图**：
- 强连通：任意两顶点间互相有有向路径
- 弱连通：忽略方向后连通

### 5.4 图的矩阵表示

#### 邻接矩阵 A（n×n）

A[i][j] = 1  若 {vᵢ,vⱼ} ∈ E（或(vᵢ,vⱼ)∈A）  
A[i][j] = 0  否则

**重要性质**：A^k[i][j] = 从vᵢ到vⱼ长度为k的通路数

> 💡 用矩阵乘法可以求多步可达性！

#### 关联矩阵 M（n×m，n个顶点m条边）

无向图：M[i][j] = 1 若顶点i关联边j，否则为0  
有向图：M[i][j] = 1（出发），-1（到达），0（不关联）

### 5.5 最短路径（Dijkstra算法）

**Dijkstra算法**：求从源点s到所有顶点的最短路径（适用于非负权重）

```
算法步骤：
1. 初始化：dist[s]=0，其余dist[v]=∞
2. 选未访问顶点中dist最小的u
3. 更新u的所有邻居v：
   if dist[u] + w(u,v) < dist[v]:
       dist[v] = dist[u] + w(u,v)
4. 标记u为已访问
5. 重复2-4直到所有顶点已访问
```

时间复杂度：O(n²)

**色数 χ(G)**：最少需要的颜色数

**四色定理**：平面图的色数 ≤ 4（任何地图用4种颜色可区分）

## 第6章：特殊的图

### 6.1 二部图（Bipartite Graph）

**定义**：顶点集可分成两个不相交子集V₁, V₂，所有边都连接V₁和V₂的顶点（无V₁内部或V₂内部的边）

**判定定理**：G是二部图 ↔ G中没有奇数长度的圈

**完全二部图 K_{m,n}**：V₁中m个顶点与V₂中n个顶点全部相连  
边数 = m×n

> 💡 **应用**：匹配问题（工作分配、相亲配对）

#### 匹配

**匹配**：边集M，使M中任意两边无公共顶点

**完美匹配**：每个顶点都被覆盖

**Hall定理**：二部图G=(V₁∪V₂, E)有完美匹配（覆盖V₁的匹配） ↔ 对V₁的任意子集S，|N(S)| ≥ |S|

---

### 6.2 欧拉图

**欧拉回路**：经过图中每条边**恰好一次**的回路（起点=终点）  
**欧拉通路**：经过每条边恰好一次但不必回到起点

#### 判定定理

| 类型 | 判定条件 |
|------|----------|
| 无向图有欧拉回路 | 连通 且 所有顶点度数为偶数 |
| 无向图有欧拉通路 | 连通 且 恰好有2个奇度顶点（这2个为起终点）|
| 有向图有欧拉回路 | 弱连通 且 每个顶点入度=出度 |
| 有向图有欧拉通路 | 弱连通 且 恰好一个顶点出度=入度+1，一个顶点入度=出度+1 |

> 💡 **经典问题**：哥尼斯堡七桥问题（欧拉1736年解决）  
> 4个地区（顶点），7座桥（边），度数均为奇数，无欧拉回路，一笔画不可能！

---

### 6.3 哈密顿图

**哈密顿回路**：经过图中每个顶点**恰好一次**的回路  
**哈密顿通路**：经过每个顶点恰好一次

⚠️ **注意区别**：欧拉图关注边，哈密顿图关注顶点

#### 判定条件（充分条件，非充要！）

**Dirac定理**：若n≥3的简单图G中每个顶点度数≥n/2，则G有哈密顿回路

> 哈密顿问题是NP完全问题，没有已知的多项式算法！

---

### 6.4 平面图

**平面图**：可以画在平面上使得边不相交（除顶点外）的图

**平面嵌入**：边不相交的平面画法

#### 欧拉公式（平面图最重要的公式！）

对连通平面图：**n - m + f = 2**

其中：
- n = 顶点数
- m = 边数  
- f = 面数（包含无界面）

**推论**：
- m ≤ 3n - 6（简单连通平面图）
- 若无三角形：m ≤ 2n - 4

**非平面图判定**：若 m > 3n-6，则不是平面图

> 💡 **典型非平面图**：K₅（5个顶点完全图）和 K_{3,3}（完全二部图）  
> **库拉托夫斯基定理**：G是平面图 ↔ G不含K₅或K_{3,3}的细分子图

### 7.1 无向树与生成树

#### 树的等价定义

以下条件等价（任选一个作定义）：
1. 连通且无圈
2. 任意两顶点间有唯一路径
3. 连通图，去掉任意一条边后不连通
4. 无圈图，加上任意一条新边恰好产生一个圈
5. n个顶点，n-1条边，连通

> 💡 记住：**树 = 连通 + 无圈 = 最少边的连通图**

#### 生成树

**生成树**：包含图G所有顶点的树（G的子图）

**任何连通图都有生成树！**

#### 最小生成树（MST）

带权图中，总权重最小的生成树

**Kruskal算法**（贪心）：
```
按权重从小到大排列所有边
依次加入边，若不形成圈则接受，否则跳过
直到选了n-1条边
```

**Prim算法**（贪心）：
```
从任意顶点开始
每次选连接已访问顶点和未访问顶点的最小权重边
直到所有顶点都在树中
```

### 7.2 根树及其应用

#### 根树基本概念

```
        r（根节点，第0层）
       / \
      a   b（第1层）
     / \   \
    c   d   e（第2层）
             \
              f（第3层）
```

| 概念 | 定义 |
|------|------|
| 根 | 没有父节点的顶点 |
| 叶（叶节点）| 没有子节点的顶点 |
| 内部节点 | 非叶节点 |
| 深度/层数 | 从根到该顶点的边数 |
| 高度 | 树中最大深度 |
| 度数 | 子节点数量 |

#### 二叉树

每个节点最多有2个子节点的根树

**满二叉树**：每个非叶节点恰好有2个子节点  
**完全二叉树**：除最后一层外其他层全满，最后一层从左到右填充

**重要性质**：
- 高度为h的二叉树最多有 2^(h+1) - 1 个节点
- n个节点的完全二叉树高度为 ⌊log₂n⌋

#### 二叉树遍历

```
     1
    / \
   2   3
  / \
 4   5

前序（根-左-右）: 1, 2, 4, 5, 3
中序（左-根-右）: 4, 2, 5, 1, 3
后序（左-右-根）: 4, 5, 2, 3, 1
```

#### Huffman树（最优前缀码）

**用途**：数据压缩（字符编码）

**思想**：高频字符用短编码，低频字符用长编码

**构造步骤**：
```
1. 以每个字符的频率为权重，建立n个单节点树
2. 选两棵权重最小的树合并（权重之和为新根的权重）
3. 重复直到只剩一棵树
4. 左子树=0，右子树=1，从根到叶的路径即为该字符的编码
```

**Huffman编码是最优前缀码**（任何编码都不是另一个的前缀）

### 8.1 基本计数原理

#### 加法原理（分类）
若完成任务有n种相互独立的方法，第i种有mᵢ种选择，则共有 m₁+m₂+...+mₙ 种方法。

#### 乘法原理（分步）
若完成任务需经过n个步骤，第i步有mᵢ种选择，则共有 m₁×m₂×...×mₙ 种方法。

> 💡 **分类用加，分步用乘！**

---

### 8.2 排列与组合

#### 排列（有序选取）

从n个不同元素中取r个的**有序**排列数：

**P(n,r) = A(n,r) = n!/(n-r)! = n×(n-1)×...×(n-r+1)**

#### 组合（无序选取）

从n个不同元素中取r个的**无序**组合数：

**C(n,r) = C(n,n-r) = n!/(r!(n-r)!)**

**常用性质：**
- C(n,0) = C(n,n) = 1
- C(n,r) = C(n,n-r)（对称性）
- C(n,r) = C(n-1,r) + C(n-1,r-1)（帕斯卡恒等式）

#### 帕斯卡三角形（杨辉三角）

```
       1
      1 1
     1 2 1
    1 3 3 1
   1 4 6 4 1
  1 5 10 10 5 1
```

每个数 = 上方两数之和 → 对应C(n,r)的值

#### 二项式定理

**(a+b)ⁿ = Σ_{r=0}^{n} C(n,r) · aⁿ⁻ʳ · bʳ**

常用：(1+x)ⁿ = Σ C(n,r) xʳ

#### 多重集合排列（重复元素的排列）

n个元素中有重复（n₁个a₁, n₂个a₂, ...），则排列数：

**n! / (n₁! × n₂! × ... × nₖ!)**（n₁+n₂+...+nₖ=n）

#### 重复组合（隔板法）

从n种元素中可重复地取r个的组合数：

**C(n+r-1, r)**

> 💡 **隔板法直觉**：把r个球放入n个盒子（可空），用n-1个隔板分割

---

### 8.3 鸽巢原理

**基本形式**：若n+1只鸽子飞入n个巢，至少有一个巢有2只以上的鸽子。

**推广形式**：若n个物品分入m组，至少有一组包含 ⌈n/m⌉ 个物品。

**例题**：任取13个人，其中至少有2人生日在同一月份。  
（13个人，12个月，13>12，鸽巢原理立刻得出）

---

### 8.4 递推方程

#### 建立递推关系

**例**：斐波那契数列：f(n) = f(n-1) + f(n-2)，f(0)=0, f(1)=1

#### 常系数线性递推方程

形式：aₙ = c₁aₙ₋₁ + c₂aₙ₋₂ + ... + cₖaₙ₋ₖ

**特征方程**：xᵏ = c₁xᵏ⁻¹ + c₂xᵏ⁻² + ... + cₖ

**求解步骤：**
1. 写出特征方程
2. 求特征根
3. 根据特征根的形式写出通解
4. 代入初始条件确定系数

| 特征根情况 | 通解形式 |
|------------|----------|
| k个不同实根 r₁,...,rₖ | aₙ = α₁r₁ⁿ + α₂r₂ⁿ + ... + αₖrₖⁿ |
| 重根r（重数m）| (α₀ + α₁n + ... + αₘ₋₁nᵐ⁻¹)rⁿ |

**例**：解递推 aₙ = 5aₙ₋₁ - 6aₙ₋₂，a₀=0, a₁=1

特征方程：x² = 5x - 6 → x² - 5x + 6 = 0  
特征根：x = 2 或 x = 3  
通解：aₙ = α·2ⁿ + β·3ⁿ  
代入初始条件：a₀=0: α+β=0，a₁=1: 2α+3β=1  
解得：α=-1, β=1  
所以：**aₙ = 3ⁿ - 2ⁿ**

### 9.1 二元运算及其性质

**二元运算**：f: A×A → A（封闭性）

#### 运算性质

| 性质 | 定义 | 例子 |
|------|------|------|
| 封闭性 | a○b ∈ A | 整数加法 |
| 交换律 | a○b = b○a | 加法、乘法 |
| 结合律 | (a○b)○c = a○(b○c) | 加法、乘法 |
| 分配律 | a○(b*c) = (a○b)*(a○c) | 乘法对加法 |

| 元素 | 定义 |
|------|------|
| 幺元（单位元）e | a○e = e○a = a，对所有a |

| 右幺元 eᵣ | a○eᵣ = a |
| 零元 θ | a○θ = θ○a = θ |
| 逆元 a⁻¹ | a○a⁻¹ = a⁻¹○a = e |

**定理**：幺元唯一；若幺元存在，则每个元素的逆元唯一

### 9.2 群（Group）

**群**的定义 <G, ○> 满足：
1. 封闭性
2. 结合律
3. 有幺元 e
4. 每个元素有逆元

**阿贝尔群（交换群）**：还满足交换律

#### 常见群的例子

| 群 | 运算 | 幺元 |
|----|------|------|
| <ℤ, +> | 加法 | 0 |
| <ℚ\{0}, ×> | 乘法 | 1 |
| <{-1,1}, ×> | 乘法 | 1 |
| <Sₙ, ∘> | 排列复合 | 恒等排列 |

#### 群的重要性质

- 消去律：a○b = a○c → b=c
- 方程 a○x=b 有唯一解 x=a⁻¹○b
- (a○b)⁻¹ = b⁻¹○a⁻¹

#### 子群

**子群**：群G的非空子集H，在G的运算下也构成群

**判定定理**：H是G的子群 ↔ 
1. H ≠ ∅
2. ∀a,b∈H: a○b∈H（封闭）
3. ∀a∈H: a⁻¹∈H（逆元）

#### 陪集与拉格朗日定理

**左陪集**：aH = {a○h | h∈H}

**拉格朗日定理**：若H是有限群G的子群，则|H|整除|G|

即：**群的阶整除子群的阶**

### 9.3 环与域

#### 环（Ring）<R, +, ·>

满足：
1. <R, +> 是阿贝尔群
2. <R, ·> 是半群（满足结合律，封闭）
3. ·对+有分配律

**例**：<ℤ, +, ×>，<ℤₙ, +ₙ, ×ₙ>（模n整数环）

#### 整环

无零因子的交换环（ab=0 → a=0或b=0）

#### 域（Field）

每个非零元都有乘法逆元的交换整环

**例**：<ℚ, +, ×>，<ℝ, +, ×>，<ℤₚ, +, ×>（p为素数）

---

### 9.4 格与布尔代数

#### 格（Lattice）

偏序集(L, ≤)，若任意两个元素a,b的**最小上界（a∨b）**和**最大下界（a∧b）**都存在，则称为格。

**格的运算性质**：
- 幂等律：a∧a=a，a∨a=a
- 交换律：a∧b=b∧a，a∨b=b∨a
- 结合律：a∧(b∧c)=(a∧b)∧c
- 吸收律：a∧(a∨b)=a，a∨(a∧b)=a

#### 布尔代数

有补有界分配格，即满足：
1. 格运算
2. 存在最大元1和最小元0
3. 分配律
4. 每个元素a有补元ā（a∧ā=0，a∨ā=1）

**重要性质（德摩根律）**：
- a∧b的补 = ā∨b̄
- a∨b的补 = ā∧b̄

**布尔代数的原子**：覆盖0的元素（最小非零元素）

**定理**：有限布尔代数 ≅ 某集合的幂集布尔代数

### 10.1 形式语言基础

#### 基本概念

- **字母表 Σ**：有限的符号集合
- **字符串**：字母表上符号的有限序列
- **空串 ε**：长度为0的字符串
- **Σ***：Σ上所有字符串的集合
- **语言 L**：L ⊆ Σ*（某些字符串的集合）

#### 形式文法

G = (V, T, P, S) 其中：
- V：变元（非终结符）集合
- T：终结符集合（V∩T=∅）
- P：产生式规则集合（α → β 的形式）
- S：开始符号（S∈V）

**推导**：反复应用产生式，从S推导出终结符串

#### 乔姆斯基层次（重要！）

| 类型 | 文法名称 | 产生式限制 | 识别机器 |
|------|----------|------------|----------|
| 0型 | 无限制文法 | α→β（α含变元）| 图灵机 |
| 1型 | 上下文相关文法 | αAβ→αγβ（γ≠ε）| 线性有界自动机 |
| 2型 | 上下文无关文法（CFG）| A→γ（左边单个变元）| 下推自动机 |
| 3型 | 正则文法 | A→aB或A→a | 有穷自动机 |

> 0型 ⊃ 1型 ⊃ 2型 ⊃ 3型（能力依次减弱，限制依次增强）

---

### 10.2 有穷自动机（FA）

#### 确定型有穷自动机（DFA）

DFA = (Q, Σ, δ, q₀, F) 其中：
- Q：有穷状态集
- Σ：输入字母表
- δ: Q×Σ → Q（转移函数，确定性！）
- q₀∈Q：初始状态
- F⊆Q：接受状态集

**工作方式**：读入字符串，按转移函数转换状态，最终状态∈F则接受

**例**：识别以0结尾的二进制串

```
状态转移图：

      0          1
→q₀ ──────→ q₁ ←───┐
            │  │     │
  1         └──┘     │
            ↑         │
            └─────────┘
    （q₁是接受状态，q₀是非接受状态）

重新画：
→(q₀) ──0──→ ((q₁))
  ↑             |
  └─────1───────┘
  (q₀)──1──→(q₀)
((q₁))──0──→((q₁))
```

#### 非确定型有穷自动机（NFA）

δ: Q×Σ → P(Q)（转移到状态的集合，可为空）

NFA比DFA表达更方便，但识别能力相同！

#### NFA→DFA（子集构造法）

将NFA的状态集合作为DFA的状态，通过ε-闭包处理ε转移。

**关键**：DFA状态是NFA状态的子集（所以DFA可能有指数级更多状态，但语言相同）

---

### 10.3 正则表达式

**正则表达式**（RE）递归定义：
- ∅（空语言）
- ε（仅含空串）
- a（单个字符，a∈Σ）
- 若r,s是RE：(r|s)，(rs)，(r*)也是RE

**对应的语言：**
- r|s：L(r)∪L(s)（或）
- rs：L(r)·L(s)（拼接）
- r*：L(r)*（Kleene闭包，0次或多次）

**定理**：正则表达式 ↔ 有穷自动机 ↔ 正则文法（三者等价！）

---

### 10.4 图灵机（Turing Machine）

**图灵机**：计算理论中最强大的计算模型

TM = (Q, Σ, Γ, δ, q₀, B, F) 其中：
- Γ：纸带字母表（Σ⊆Γ）
- B：空白符
- δ: Q×Γ → Q×Γ×{L,R}（状态，写什么，左/右移）

**重要性质：**
- 图灵机可以识别所有能用算法解决的问题（Church-Turing论题）
- 停机问题（Halting Problem）是不可判定的！

# 💡 经典例题精讲汇总

## 逻辑类

**【例题1】** 证明：¬p∧(p∨q) ⊢ q

| 步骤 | 公式 | 理由 |
|------|------|------|
| 1 | ¬p | 化简（前提¬p∧(p∨q)，取左合取） |
| 2 | p∨q | 化简（取右合取） |
| 3 | q | DS（步骤1,2：析取三段论） |

## 集合类

**【例题2】** 设|A|=3，求|P(P(A))|

解：|A|=3 → |P(A)|=2³=8 → |P(P(A))|=2⁸=**256**

## 图论类

**【例题3】** 一个6顶点的图有9条边，证明它不是平面图。

解：由推论，简单连通平面图满足 m ≤ 3n-6  
这里 n=6, m=9  
检验：3×6-6 = 12 ≥ 9  
（不能直接排除，需更多分析或用另一推论）

若图无三角形，则 m ≤ 2n-4 = 2×6-4 = 8 < 9  
故若该图无三角形，则**不是平面图**。

## 组合类

**【例题4】** 斐波那契数列 f(n)=f(n-1)+f(n-2)，f(1)=1, f(2)=1，求通项公式

特征方程：x² = x + 1 → x² - x - 1 = 0  
特征根：x = (1±√5)/2

设 φ = (1+√5)/2（黄金比例），ψ = (1-√5)/2

通解：f(n) = αφⁿ + βψⁿ  
代入：f(1)=1, f(2)=1  
解得：α = 1/√5, β = -1/√5

**通项公式：f(n) = (φⁿ - ψⁿ)/√5**（比内公式）
